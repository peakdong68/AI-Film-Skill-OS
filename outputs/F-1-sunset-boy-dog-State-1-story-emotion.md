## STATE 1 — 故事与情绪设计
### 项目：F-1 日落·男孩·狗

---

## 一、叙事结构图 — 三幕结构

```
第一幕：孤独（0s-15s）— 建立世界与角色现状
  转折点 1：流浪狗的出现 — 打破男孩的独处

第二幕：试探与连接（15s-45s）— 逐渐升级的信任
  中点转折：男孩掰开面包 — 不可逆的善意举动
  转折点 2：小狗第一次犹豫 — 信任的至暗瞬间

第三幕：陪伴（45s-60s）— 温暖的解决
```

---

## 二、场景拆解图

### 场景 1：日落独处（0s-10s）

| 条目 | 内容 |
|------|------|
| **场景目的** | 建立 | 引入角色、时间、空间、情绪基调 |
| **情绪功能** | 温和的孤独感，引起同情 |
| **叙事功能** | 建立世界观与角色现状 |
| **涉及角色** | 男孩 |
| **冲突类型** | 内在（孤独） |
| **入场状态 → 离场状态** | 赶路中 → 安顿坐下 |

**视觉摘要**：日落时分，男孩背着书包走在回家路上，找了一处台阶/长椅坐下，放下书包和水瓶。

---

### 场景 2：相遇（10s-20s）

| 条目 | 内容 |
|------|------|
| **场景目的** | 推进 | 激励事件：一次偶然的相遇 |
| **情绪功能** | 好奇+轻微不确定性——观众和男孩同时发现狗 |
| **叙事功能** | 推动情节——引入冲突/机遇 |
| **涉及角色** | 男孩、小黄狗 |
| **冲突类型** | 人际（陌生生物之间的信任试探） |
| **入场状态 → 离场状态** | 准备吃面包 → 注意到狗的存在 |

**视觉摘要**：男孩从书包里拿出面包，正要吃时眼角余光瞥到一只小黄狗在不远处望着他。

---

### 场景 3：试探与分享（20s-35s）

| 条目 | 内容 |
|------|------|
| **场景目的** | 升级 | 建立两个角色之间的信任桥梁 |
| **情绪功能** | 紧张→温暖——小狗犹豫后退→男孩耐心等待→小狗靠近 |
| **叙事功能** | 推进情节——关键行动发生 |
| **涉及角色** | 男孩、小黄狗 |
| **冲突类型** | 人际（信任的建立） |
| **入场状态 → 离场状态** | 狗保持距离 → 狗靠近并接受面包 |

**视觉摘要**：男孩温柔地看着狗，慢慢将面包掰成两半，把一半放在地上。狗犹豫，后退一步，又慢慢上前嗅了嗅，开始吃。男孩微笑。

---

### 场景 4：靠近与连接（35s-50s）

| 条目 | 内容 |
|------|------|
| **场景目的** | 揭示 | 信任达成——情感连接的建立 |
| **情绪功能** | 温暖、感动——心脏融化时刻 |
| **叙事功能** | 揭示角色——男孩的善良本质；揭示狗的状态（渴望陪伴） |
| **涉及角色** | 男孩、小黄狗 |
| **冲突类型** | 内在→人际（隔阂消失） |
| **入场状态 → 离场状态** | 狗在吃面包 → 狗坐在男孩身边，头靠在他膝盖上 |

**视觉摘要**：吃完面包后，狗没有走。它犹豫地走近，坐下，然后慢慢把头靠在男孩膝盖上。男孩愣了一下，然后轻轻抚摸狗的头。

---

### 场景 5：陪伴（50s-60s）

| 条目 | 内容 |
|------|------|
| **场景目的** | 收束 | 情绪沉淀——叙事完成 |
| **情绪功能** | 宁静的满足、治愈——情绪释放后的安宁 |
| **叙事功能** | 完成弧线——两个孤独的灵魂找到彼此 |
| **涉及角色** | 男孩、小黄狗 |
| **冲突类型** | 无（已解决） |
| **入场状态 → 离场状态** | 刚建立连接 → 沉浸在彼此的陪伴中 |

**视觉摘要**：一人一狗并排而坐，男孩抚摸狗，书包和水瓶安静地放在旁边。夕阳在背景中缓缓沉下，影子拉长。

## 三、因果链

```
场景 1：男孩独坐，准备吃面包
    ↓（孤独暗示——吸引观众同理心）
场景 2：男孩注意到流浪狗
    ↓（相遇是偶然，但连接是选择）
场景 3：男孩主动掰开面包分享
    ↓（善意引发信任）
场景 4：狗靠近，把头靠在男孩膝盖上
    ↓（信任建立后，情感外化）
场景 5：一人一狗并排而坐，陪伴的宁静
    ↓（叙事闭合：孤独→陪伴）
```

**因果验证**：移除任意一环，下一环都不成立。链条稳固。

## 四、导演意图层

| 场景 | 为什么存在？ | 结束改变了什么？ | 观众带什么情绪离开？ |
|------|-------------|-----------------|---------------------|
| 1. 日落独处 | 建立男孩孤独的日常状态，为后面对比做铺垫 | 观众知道了"这个男孩独自一人" | 略带酸楚的安静同情 |
| 2. 相遇 | 引入激励事件，启动叙事 | 从"独处"变为"有了可能互动的对象" | 好奇——会发生什么？ |
| 3. 试探与分享 | 展示男孩的善良是主动选择的（而非被动） | 从"陌生人"变为"分享者" | 温暖——被善良打动 |
| 4. 靠近与连接 | 呈现善意得到回应的时刻——情感峰值 | 从"怀疑"变为"信任" | 感动——心脏融化 |
| 5. 陪伴 | 完成情感弧线，让余韵沉淀 | 从"孤独"变为"有人陪伴" | 宁静治愈——长久的余味 |

## 五、情绪弧线设计

### 1. 情绪弧线图

```
[平静的孤独]            [好奇]            [温暖]            [感动]            [宁静的满足]
    │                    │                 │                 │                 │
  场景 1              场景 2             场景 3            场景 4           场景 5
    ↓                    ↓                 ↓                 ↓                 ↓
   3/10                5/10              6→7/10           8/10              5/10
```

### 2. 情绪节拍

| 节拍 | 时间 | 事件 | 情绪转变 |
|------|------|------|---------|
| 节拍 1 | 10s | 男孩注意到狗 | 孤独 → 好奇 |
| 节拍 2 | 20s | 男孩掰开面包 | 好奇 → 温暖 |
| 节拍 3 | 35s | 狗吃完面包没走 | 温暖 → 期待 |
| 节拍 4 | 42s | 狗把头靠在男孩膝盖上 | 感动（峰值） |
| 节拍 5 | 50s | 男孩抚摸狗，并坐 | 感动 → 宁静满足 |

### 3. 强度时间线

```
0s ──── 15s ──── 25s ──── 35s ──── 50s ──── 60s
 3        5        6        8         6       5
[独处]   [发现]   [试探]   [信任]   [陪伴]  [余韵]
                    ↑                    ↑
              掰面包时刻          头靠膝盖（峰值）
```

### 4. 角色情绪变化

**男孩：**
- 场景 1：平静的孤独 → 强度 3（日常状态，习惯了一个人）
- 场景 2：好奇 → 强度 5（发现狗，有点意外）
- 场景 3：温暖 + 小心 → 强度 6（主动分享，展现善意）
- 场景 4：惊讶 → 感动 → 强度 8（狗靠过来的瞬间，心被触动）
- 场景 5：满足、安宁 → 强度 5（不再孤独了）
- **弧线：从习惯孤独 → 体验到陪伴的温暖**

**小黄狗：**
- 场景 2：警觉、谨慎 → 强度 6（流浪狗的生存本能）
- 场景 3：犹豫 → 试探 → 接受 → 强度 7→4（饥饿战胜恐惧）
- 场景 4：信任 → 亲近 → 强度 3（吃饱后渴望情感接触）
- 场景 5：安全、放松 → 强度 2（找到了暂时的归属）
- **弧线：从警惕流浪 → 获得片刻的安全与温暖**

### 5. 情绪→视觉映射

| 情绪 | 摄影机 | 灯光 | 色彩 | 节奏 |
|------|--------|------|------|------|
| 孤独（3） | 远景/中景，固定镜头，负空间构图 | 金色夕阳但留有大片阴影 | 暖金+暗蓝（冷暖对比） | 缓慢，呼吸感，6s+镜头 |
| 好奇（5） | 中景切近景，轻微推镜 | 逆光勾勒轮廓 | 金色主调 | 中速，8-10s镜头 |
| 温暖（6-7） | 中近景对切，过肩镜头 | 暖光柔和，光晕 | 橙黄+暖棕 | 中速偏缓，10s镜头 |
| 感动（8） | 特写近景，缓慢推镜 | 黄金时刻的暖光包裹 | 金黄+暖橙（峰值） | 缓慢，12s+长镜头 |
| 宁静（5） | 中景→远景，拉镜 | 光线渐柔，进入暮色 | 暖橙转向紫蓝 | 极缓，10s释放 |
