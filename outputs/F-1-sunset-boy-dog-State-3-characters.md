## 角色 A：男孩

### 1. 角色身份核心

| 条目 | 内容 |
|------|------|
| **角色名** | 男孩（无名——代表每一个普通的善良孩子） |
| **叙事角色** | 主角 |
| **年龄** | 10-12 岁 |
| **性别表达** | 男孩 |
| **情绪身份** | 安静、敏感、习惯独处但渴望连接 |
| **心理画像** | 一个习惯独自放学的孩子，书包里装着课本和水瓶。他不是不开心的——只是安静。面对流浪狗时，他的第一反应不是害怕，而是好奇，然后是纯粹的善意。他的善良是本能性的，不需要思考。 |
| **故事功能** | 代表善意本身——在一个日常时刻做出分享的选择 |
| **角色弧线** | 习惯独自一人 → 体验陪伴的温暖 —— 他不知道自己需要陪伴，直到小狗靠过来 |

| 条目 | 内容 |
|------|------|
| **脸型** | 鹅蛋脸，略带婴儿肥，线条柔和 |
| **肤色** | 浅小麦色，健康，带着放学后的微红 |
| **眼型** | 单眼皮，略微下垂的眼角，安静的眼神 |

| **下颌** | 柔和圆润，孩童未完全褪去的曲线 |

| **体态** | 精瘦，肩膀窄但直立 |

## 角色 B：小黄狗

### 1. 角色身份核心

| 条目 | 内容 |
|------|------|
| **角色名** | 小黄狗（流浪狗——无主） |
| **叙事角色** | 配角（次要主角，触发情感反应的角色） |
| **年龄** | 约 2-3 岁，年轻成年犬 |
| **品种印象** | 中华田园犬（土狗），非纯种 |
| **情绪身份** | 谨慎、饥饿、渴望温暖但害怕人类——流浪的生存本能在与对陪伴的渴望之间摇摆 |
| **心理画像** | 它已在街头生活了一段时间。它学会了保持距离——人类有时会喂它，有时会赶它。它近距离观察这个男孩：他和小狗分享面包的安静举动让它决定冒一次险。它的靠近不是乞讨，而是信任的试探。 |
| **故事功能** | 触发男孩善意的对象——同时也代表所有渴望被善待的流浪生命 |
| **角色弧线** | 警惕流浪 → 试探信任 → 获得温暖 —— 它不是被驯服的，它是被善意打动的 |

### 2. 视觉身份系统

#### 头部/面部系统

| 条目 | 内容 |
|------|------|
| **脸型** | 三角形脸，吻部中等偏长 |
| **毛色** | 浅黄色/淡棕色（#D4A373 → #B8860B），腹部和脸颊略浅 |
| **眼型** | 圆润的杏眼，眼神柔和但带着警觉 |
| **眼睛颜色** | 深褐色，眼周有一圈深色毛 |
| **耳朵** | 立耳，略微向前倾——警觉时会竖得更直，放松时会自然向后 |
| **鼻子** | 黑色湿润的鼻头 |
| **嘴型** | 安静时嘴闭合，喘气时会微微露出舌头 |
| **辨识特征** | 右耳尖有一个小缺口（流浪生活的印记）；嘴角有一点天生的深色毛 |

#### 毛发系统

| 条目 | 内容 |
|------|------|
| **毛型** | 短毛，光滑贴服 |
| **长度** | 全身均匀约 2-3cm |
| **颜色** | 背部浅棕黄（#BC6C25），胸腹和脸颊淡黄（#EECBA1），尾巴尖略深 |
| **质地** | 健康但有流浪痕迹——部分区域毛略粗糙 |
| **特殊标记** | 胸口有一小块白色毛（不规则形状，约巴掌大） |

#### 体型系统

| **姿势** | 站立时耳朵竖起，尾巴低垂（不夹尾但也不翘）；坐下时前腿并拢 |
| **动作签名** | 走路时轻盈但谨慎，步伐安静；尾巴在放松时会轻微摆动；接近时头放低，身体略侧 |

### 3. 行为系统

#### 眼神逻辑

| 角色 | 默认 | 警惕时 | 信任时 |
|------|------|--------|--------|
| **男孩** | 看远处/低头，安静的眼神 | — | 温和地直视，不闪躲 |
| **小黄狗** | 扫视周围，快速视线移动 | 盯视但头略低，不敢直视 | 眼神放松，半闭眼（安全） |

#### 情绪→动作映射

| 情绪 | 男孩 | 小黄狗 |
|------|------|--------|
| **孤独** | 坐着时，无目的地看着远方，手指轻轻敲腿 | — |
| **好奇** | 头微侧，目光追随狗 | 耳朵前倾，头歪一下，前爪交替小步移动 |
| **犹豫** | 拿面包的手在半空中停一下 | 后退一步，回头看，前爪抬起又放下 |
| **温暖/喜悦** | 嘴角微扬，笑时眼睛眯起 | 耳朵向后放松，尾巴小幅度摆动 |
| **感动** | 呼吸变慢，动作暂停片刻，眼眶微亮 | 头低下靠过去，闭眼，耳朵完全放松 |
| **满足** | 姿势松弛，肩膀垂下 | 趴下/靠紧，半闭眼，气息平稳 |

## 三、多角色系统

### 角色关系图

```
[男孩] ←———→ [小黄狗]
   ↑              ↑
   善意分享     谨慎信任
```

- **关系类型**：陌生人→分享者→陪伴
- **权力动态**：男孩有食物=主导；但男孩不使用这种权力，而是平等分享
- **情绪基调**：温暖、试探性、最终达到宁静的陪伴

### 视觉对比设计

| 维度 | 男孩 | 小黄狗 |
|------|------|--------|
| **体型对比** | 直立（150cm vs 45cm 肩高） | 四足/坐姿，矮小 |
| **色彩对比** | 白色+深蓝（人造成的颜色） | 黄褐色（自然的颜色） |
| **形状语言** | 直线条（站立/坐姿的轮廓） | 曲线条（狗的轮廓弧线） |
| **动静对比** | 相对静止，动作克制 | 小的移动（尾巴、耳朵、头） |

### 互动规则

- **物理距离规范**：场景 2-3 保持 ~1.5m 距离；场景 4-5 缩小到 0-0.3m
- **眼神接触模式**：场景 2 回避→间接→场景 3 交替→场景 4-5 直接温和
- **触碰边界**：男孩不抓不抱——只伸手（手背）让狗嗅，然后轻抚头/背
- **反应规则**：任何突然的动作会导致狗后退——男孩理解这一点，故意放慢所有动作

## 四、连续性锁定系统

### 硬锁（不可变）

| 序号 | 角色 | 锁定项 |
|------|------|--------|
| ✅ | 男孩 | 鹅蛋脸 + 单眼皮 + 深褐眼 + 短发自然黑 + 白T恤深蓝短裤 |
| ✅ | 男孩 | 右脸颊小痣 |
| ✅ | 小黄狗 | 浅黄短毛 + 立耳 + 右耳尖缺口 + 胸口白斑 |
| ✅ | 小黄狗 | 中等体型瘦削 + 棕色眼 + 黑色鼻头 |

### 软变化（允许）

- 日落光照下肤色/毛色会变成暖金色调
- 情绪导致的面部/耳朵变化（在情绪→动作映射范围内）
- 不同镜头角度下的面部呈现
- 被风吹起的刘海/狗毛

### 禁止变化

- ❌ 男孩换装（必须永远是白T恤+深蓝短裤）
- ❌ 狗毛色变为深棕或黑色
- ❌ 男孩发型变为长发或碎发
- ❌ 狗胖了或瘦了（保持流浪体态）
- ❌ 男孩/狗的年龄或体型出现跳变
