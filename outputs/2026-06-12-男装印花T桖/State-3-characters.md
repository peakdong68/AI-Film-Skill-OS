## STATE 3 — 角色锁定

### 角色 1：男模（产品展示者）

#### 1. 角色身份核心

| 项目 | 内容 |
|------|------|
| **叙事角色** | 产品展示者（Product Presenter） |
| **年龄范围** | 22–28 |
| **性别表达** | 男性 |
| **情绪身份** | 自信、松弛、微酷——不笑时冷感，微表情带松弛感 |
| **心理画像** | 都市年轻男性，穿搭有自我主张，不刻意耍帅 |
| **故事功能** | 承载印花T桖上身效果，连接产品与消费者向往感 |
| **角色弧线** | 出场（亮相）→ 展示（细节）→ 定格（最佳状态） |

#### 2. 视觉身份系统

**面部系统：**

| 参数 | 锁定值 |
|------|--------|
| 脸型 | 略长偏棱角，下颌线清晰 |
| 肤色 | 亚洲肤色，中性偏暖调 |
| 眼型 | 内双或单眼皮，眼型偏长 |
| 眼珠颜色 | 深棕 |
| 眉毛 | 自然浓密，微直，不刻意修 |
| 鼻子 | 鼻梁直挺，鼻尖适中 |
| 嘴型 | 嘴唇适中偏薄，自然闭合 |
| 下颌/下巴 | 线条清晰不过于尖锐 |
| 辨识特征 | 干净面部，无明显疤痕/痣 |
| 面部结构 | 骨相清晰，适合侧光/逆光勾勒轮廓 |

**发型系统：**

| 参数 | 锁定值 |
|------|--------|
| 发型 | 韩式微卷短发 / 碎盖短发 |
| 长度 | 额前刘海可触眉，两侧推短 |
| 颜色 | 自然黑或深棕 |
| 质地 | 微蓬松，自然纹理 |
| 分线方向 | 四六分或中分自然散落 |
| 动态质感 | 走路时轻微晃动，保持纹理不乱 |

**体型系统：**

| 参数 | 锁定值 |
|------|--------|
| 体型 | 精瘦偏运动——肩宽腰窄 |
| 身高 | 178–185cm（视觉比例） |
| 体态 | 偏精瘦，有轻微肌肉线条 |
| 姿势 | 自然挺拔，不僵硬不驼 |
| 动作签名 | 走路带轻微跨步节奏，手自然插兜或垂放 |

**服装系统：**

| 参数 | 锁定值 |
|------|--------|
| 标志服装 | 印花T桖 + 简约下装（黑/白/牛仔） |
| 配色方案 | 上衣（印花主题色）+ 下装（黑/白/牛仔蓝） |
| 面料语言 | T桖：棉质/混纺；下装：牛仔/棉质 |
| 版型 | T桖宽松或标准合身 |
| 配饰 | 极简——可选细链项链/简约手表 |

#### 3. 镜头行为系统

**情绪→动作映射：**

| 场景 | 情绪 | 面部 | 身体 | 眼神 |
|------|------|------|------|------|
| S1 Hook | 冷静自信 | 微抿嘴、下颌微抬 | 迎面走向镜头 | 直视镜头 |
| S2 细节 | —（局部） | 不在画内/局部 | 上身微转展示印花 | — |
| S3 轮换 | 轻松切换 | 微表情随印花变化 | 自然站姿变化 | 跟随镜头方向 |
| S4 Hero | 自信定格 | 嘴角微扬 | 侧身/微仰 | 偏上远方 |
| S5 CTA | —（不在画内） | — | — | — |

### 角色 2：印花T桖（产品身份）

| 项目 | 锁定值 |
|------|--------|
| **品类** | 男装短袖印花T桖 |
| **版型** | 标准合身 / 微宽松落肩 |
| **面料** | 纯棉或棉混纺 |
| **领型** | 圆领，罗纹领口 |
| **印花工艺** | 数码印花/胶印/刺绣 |
| **印花位置** | 胸前/后背/满版 |

### 连续性锁定

**硬锁：**
- 面部——脸型、五官比例
- 发型——韩式微卷短发、深色
- 体型——精瘦运动、肩宽腰窄
- 服装——印花T桖（版型一致）+ 纯色下装

**软变化：** 面部光照角度、微表情、站姿微调、印花图案
