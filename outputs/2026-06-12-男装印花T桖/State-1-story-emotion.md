## STATE 1 — 故事与情绪设计

### 1. 叙事结构图（产品展示模式）

```
第一幕：Hook 建立 (0–3s)
  模特亮相 + 第一印花展示 — 建立产品视觉锚点
    ↓
第二幕：展开展示 (3–11s)
  印花细节特写 (3–7s) → 多款印花轮换 (7–11s) — 层层递进产品信息
    ↓
第三幕：收束 + CTA (11–15s)
  Hero Shot 最佳造型定帧 → Logo/CTA 号召行动
```

### 2. 场景拆解图

| | 时间 | 标题 | 场景目的 | 功能 | 模特 |
|---|---|---|---|---|---|
| **S1** | 0–3s | Hook·亮相 | 建立 | 抓注意力 | ✅ |
| **S2** | 3–7s | 细节·印花 | 揭示 | 推进产品信息 | ✅（局部） |
| **S3** | 7–11s | 轮换·多款 | 升级 | 制造丰富感 | ✅ |
| **S4** | 11–14s | Hero·最佳造型 | 对比 | 情绪峰值 | ✅ |
| **S5** | 14–15s | CTA·收束 | 收束 | 转化驱动 | ❌ |

### 3. 因果链

```
模特亮相 → 印花吸引视线 → 多款展示证明产品线丰富 → Hero造型建立购买欲望 → CTA 触发行动
```

### 4. 导演意图层

| 场景 | 为什么存在 | 改变了什么 | 观众带走的情绪 |
|------|-----------|-----------|--------------|
| S1 Hook | 前 2s 决定划走还是停留 | 观众从"刷视频"→"停下来看" | 好奇、眼前一亮 |
| S2 细节 | 证明产品质量，消除购买疑虑 | 从"好看"→"品质可信" | 欣赏、想要 |
| S3 轮换 | 展示SKU丰富度 | 从"一件不错"→"还有更多选择" | 兴奋、选择感 |
| S4 Hero | 建立品牌/穿搭记忆点 | 从"看看"→"想拥有" | 自信、向往 |
| S5 CTA | 把情绪转化为行为 | 从"想要"→"去下单" | 行动冲动 |

### 5. 情绪曲线

```
时间轴:  0% ──── 20% ──── 47% ──── 73% ──── 93% ──── 100%
         0s      3s      7s      11s     14s     15s

强度:     7  →    6   →    8   →    9   →    7

情绪:   [Hook]  [细节]  [轮换]  [Hero]  [CTA]
        好奇    欣赏    兴奋    渴望    行动
```

### 6. 情绪节拍（关键转折）

| 节拍 | 触发事件 | 情绪转变 |
|------|---------|---------|
| **T1** | 模特上身亮相，印花图案跃出画面 | 无感 → 好奇 (7) |
| **T2** | 印花纹理特写，面料细节放大 | 好奇 → 欣赏 (6) |
| **T3** | 快速切换多款不同印花 | 欣赏 → 兴奋 (8) |
| **T4** | Hero 造型定帧，逆光/氛围光 | 兴奋 → 渴望 (9) |
| **T5** | Logo 出现 + CTA 文案 | 渴望 → 行动冲动 (7) |

### 7. 节奏结构

```
[快 — 抓注意力] → [慢 — 欣赏细节] → [极快 — 多款切换] → [缓停 — Hero定帧] → [收 — CTA]
   0-3s              3-7s                7-11s               11-14s            14-15s
   ~4 cut            ~2 cut              ~6-7 cut            1 shot            1 frame
```

| 情绪 | 摄影机方向 | 灯光方向 | 色彩方向 |
|------|-----------|---------|---------|
