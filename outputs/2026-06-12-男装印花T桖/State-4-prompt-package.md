# 男装印花T桖 · 电商带货 — 短片提示包

## 项目元数据

| 项目 | 内容 |
|------|------|
| **总时长** | 15s |
| **画幅比例** | 9:16 竖屏 |
| **视觉风格** | 商业时尚 — 干净利落、高信息密度 |
| **目标平台** | TK 短视频 |
| **导演模式** | Commercial（商业导演） |

## 一、叙事概览

### 故事梗概

一条 15 秒男装印花 T 桖产品展示短片。前 3s 模特快步亮相抓注意力；3-11s 印花细节特写+多款轮换展示 SKU 丰富度；11-14s Hero Shot 最佳造型定格营造向往感；最后 1s CTA 收束转化。冷→暖色调推进，快节奏鼓点驱动。

### 叙事结构

```
第一幕：Hook 建立 (0–3s) — 模特亮相 + 印花第一印象
第二幕：展开展示 (3–11s) — 细节特写 → 多款轮换
第三幕：收束 CTA (11–15s) — Hero 定帧 → 购买引导
```

### 情绪弧线

```
好奇(7) ──→ 欣赏(6) ──→ 兴奋(8) ──→ 渴望(9) ──→ 行动(7)
  0s         3s          7s         11s         14s
```

## 二、分镜设计方案

### Part 1：全片 (0–15s)

#### SHOT 01：Hook · 模特亮相 (0–3s)

| 维度 | 内容 |
|------|------|
| 时长 | 3s |
| 场景 | 影棚/纯色背景 |
| 目的 | 建立 — 抓注意力 |
| 视觉 | 男模穿第一款印花T桖迎面走向镜头。上方留白10%，模特占画面中下2/3。冷白背景，高反差 |
| 动作 | 从画面中远处迎面走向镜头，步幅中等，肩展开，近镜微转头亮相 |
| 摄影机 | 中全景→中近景，平视→微仰，快推，深→浅景深 |
| 灯光 | 左前上硬光主光，右侧柔光补面，轻微背轮廓光，深锐阴影，冷白调 |
| 情绪 | 无感→好奇 (7) |
| 身份 | 男模 Male-Model-001 硬锁；白色印花T桖+黑色宽腿裤 |

#### SHOT 02：印花细节 (3–7s)

| 维度 | 内容 |
|------|------|
| 时长 | 4s |
| 场景 | 同上 |
| 目的 | 揭示 — 展示纹理质感 |
| 视觉 | 大特写/特写印花局部，占画面60-80%。上身局部或平铺特写 |
| 动作 | 镜头平移滑过印花表面，模特上身微转展示不同角度 |
| 摄影机 | 大特写/特写，俯拍或正面，平移，浅景深 |
| 灯光 | 环形柔光均匀铺展，下方反光板补纹理，中性色温，面料本色 |
| 情绪 | 好奇→欣赏 (6) |
| 身份 | 产品身份：面料质感/印花清晰度锁定 |

#### SHOT 03：多款轮换 (7–11s)

| 维度 | 内容 |
|------|------|
| 时长 | 4s |
| 场景 | 同上 |
| 目的 | 升级 — SKU丰富度 |
| 视觉 | 模特更换3-4款印花T桖，每款约1s跳切。同位置保持重心稳定 |
| 动作 | 模特自然站姿微调——微转体/侧身/插兜 |
| 摄影机 | 中近景→特写交替，多角度混切，跳切 |
| 灯光 | 每款微调灯光氛围，快速切换，整体反差拉高 |
| 情绪 | 欣赏→兴奋 (8) |
| 身份 | 男模硬锁持续；多款印花T桖（版型一致） |

#### SHOT 04：Hero Shot · 最佳造型 (11–14s)

| 维度 | 内容 |
|------|------|
| 时长 | 3s |
| 场景 | 同上 |
| 目的 | 对比 — 情绪峰值，建立向往感 |
| 视觉 | 模特穿最佳印花款，居中构图，浅景深虚化。逆光勾勒轮廓，暖金琥珀调 |
| 动作 | 侧身微仰，下颌微扬，嘴角微扬，眼神偏上远方 |
| 摄影机 | 中近景，微仰，慢推，浅景深 |
| 灯光 | 逆光主导+正面柔补，银色反光板，强金色逆光，柔暖阴影 |
| 情绪 | 兴奋→渴望 (9) |
| 身份 | 男模 Hero 姿态硬锁；最佳印花锁定 |

#### SHOT 05：CTA · 收束 (14–15s)

| 维度 | 内容 |
|------|------|
| 时长 | 1s |
| 场景 | 纯色/渐变背景 |
| 目的 | 收束 — 引导购买 |
| 视觉 | Logo+产品缩略图+CTA文案，居中排版 |
| 动作 | 图形/文字静态 |
| 摄影机 | 定帧，平视 |
| 灯光 | 均匀平光，品牌色 |
| 情绪 | 渴望→行动 (7) |
| 身份 | 品牌/产品视觉一致性 |

## 三、摄影机语言规范

| 运动 | 叙事意图 | 使用镜头 |
|------|---------|---------|
| 快推 | 注意力抓取 | S1 |
| 平移 | 纹理展开 | S2 |
| 跳切 | 节奏加速 | S3 |
| 慢推 | 情绪累积 | S4 |
| 定镜 | 信息锚定 | S5 |

**构图**：竖屏安全区（头10%+底15%），中轴线稳定，重心不跳

## 四、灯光与色彩脚本

| 阶段 | 主导色 | 色温 | 反差 |
|------|--------|------|------|
| S1 Hook | 冷白+深灰 | 冷 | 高 |
| S2 细节 | 面料本色 | 中性 | 中低 |
| S3 轮换 | 多色混合 | 混搭 | 高 |
| S4 Hero | 暖金+琥珀 | 暖 | 中 |
| S5 CTA | 品牌色 | 品牌 | 平 |

**主光方向**：左前上方（S4 逆光除外）  
**色温弧线**：冷→中性→混搭→暖→品牌色

## 五、声音设计方向

- **风格**：轻电子/Urban/Lo-fi Hip-hop，BPM 100-120
- **结构**：强鼓点起(0-2s)→稳定节奏(2-7s)→加速(7-11s)→缓降(11-14s)→静默(14-15s)
