# STATE 1: 故事与情绪设计 — 面包与小狗

## 一、叙事结构 — 三幕微剧情

```
ACT I — 饥饿与注视 (0s–5s)
  场景1: 日落时分，男孩取出面包，准备咬下第一口
  场景2: 小狗的目光——悲伤而充满希望地凝视面包
  → 建立困境：饥肠辘辘的小狗，面包只有一块

ACT II — 分享的决定 (5s–10s)
  场景3: 特写——男孩将面包掰成两块，面包屑飘落
  场景4: 男孩向小狗递出半块面包
  场景5: 小狗犹豫片刻，上前轻轻接走
  → 转折点：善意战胜自私，分享发生

ACT III — 陪伴的回馈 (10s–15s)
  场景6: 小狗坐在男孩身边，男孩温柔抚摸它
  场景7: 小狗将头靠在男孩膝盖上——无声的信任
  场景8: 一人一狗并排而坐，书包和水瓶静静陪伴
  → 收束：分享换来陪伴，两个孤独的灵魂相互取暖
```

