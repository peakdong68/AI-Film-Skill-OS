## STATE 2 — 视觉设计
### 项目：F-1 日落·男孩·狗

---

## 一、导演人格锁定：Observer（旁观者）

### 视觉哲学
摄影机观察。它从不干预。它只是见证。世界自然展开，导演不操控。

| 维度 | Observer 特征 | 对本项目的意义 |
|------|--------------|--------------|
| **摄影机** | Static、Wide shot、长镜头、最小运动 | 镜头保持距离，让男孩和狗之间的互动自然流露 |
| **角度** | Eye-level，偶尔远距离 | 以平视尊重角色的世界，不附加多余的情绪解读 |
| **运动** | Static > slow tracking > 极少 Dolly | 仅在需要时缓慢移动，大多数时候固定让画面呼吸 |
| **节奏** | Slow cinema——长镜头、呼吸空间、静默 | 给每一个互动留足时间，让情感自然沉淀 |
| **光影** | 仅自然光和情节内光源 | 完全依赖黄金时刻的日光，不加人工干预 |
| **情绪** | 含蓄、暗示、从不强迫 | 观众自己发现感动，而非被镜头引导 |

### 叙事哲学
信息被展示，不被讲述。观众自行发现连接——我们不需要告诉观众"这是感人的"，我们只是记录这一刻。

## 二、摄影机语言系统

### 镜头类型语法（核心词汇）

| 类型 | 在本片中的叙事功能 |
|------|------------------|
| **远景（WS）** | 场景 1 和 5 的主要镜头——建立孤独感、空间关系、日落环境 |
| **中景（MS）** | 场景 2-4 的核心镜头——承载叙事动作、记录互动过程 |
| **中近景（MCS）** | 展示面部表情和狗的细微反应——情绪外化的窗口 |
| **特写（CU）** | 谨慎使用——仅在关键瞬间（掰面包、狗靠头、抚摸的手） |
| **过肩（OTS）** | 场景 3 中男孩和狗的视线交流——建立关系空间 |

### 运动语法

| 运动 | 使用场景 | 情绪功能 |
|------|---------|---------|
| **定镜** | 全片主导（~80% 镜头） | 观察者姿态——让动作在画面中自然发生，镜头不干涉 |
| **缓推（slow dolly-in）** | 场景 4 狗靠头瞬间 | 极慢、不被察觉的靠近——情绪的微妙深化 |
| **拉远（slow dolly-out）** | 场景 5 结尾 | 情绪抽离、融入环境——叙事闭合 |

### 角度哲学

| 角度 | 使用 |
|------|------|
| **平视（Eye-level）** | 全片 ~90%——以男孩或狗的身高视角观察世界 |
| **俯视（High-angle）** | 场景 1 独坐时一次——强化孤独感（但保持克制） |

### 焦点语法

| 焦点 | 使用 |
|------|------|
| **深焦（Deep focus）** | 场景 1、5——男孩与环境同框，展示人与世界的关系 |
| **浅景深（Shallow DOF）** | 场景 3-4——当两个角色之间产生连接时，隔离他们于背景之上 |

## 四、空间构图系统

### 取景规则

| 构图 | 使用场景 |
|------|---------|
| **三分法** | 全片默认构图——自然、不刻意 |
| **负空间** | 场景 1、5——男孩/一人一狗置于画面一侧，另一侧留给落日余晖 |
| **居中构图** | 场景 4 靠头瞬间——仪式感的对称，这是全片唯一的"重要时刻" |
| **框中框** | 场景 1 可尝试——树影/建筑框住男孩，暗示孤独 |

### 深度设计
- 每镜头维持三层深度：前景（草/地面/书包）→ 中景（角色）→ 背景（日落天空/树木/建筑）
- 角色之间的物理距离 = 情感距离（场景 2 时距离远 → 场景 4-5 时紧靠）

### 空间连续性
- 方向一致性：男孩始终坐在画面的同一个朝向（左→右/右→左统一）
- 180 度规则：保持在同一轴线拍摄，不越轴
- 空间地理：书包和水瓶的位置一旦放置，在所有镜头中保持不变

## 五、逐镜头摄影机设计

### 场景 1：日落独处（0s-10s）

| 条目 | 内容 |
|------|------|
| **镜头 1a** | 远景（WS），固定镜头，平视，深焦。男孩从画右走入，在台阶坐下。负空间构图——男孩偏左，右侧留白给落日天空。 |
| **镜头 1b** | 中景（MS），固定镜头。男孩放下书包和水瓶，深吸一口气。三分法构图。 |

### 场景 2：相遇（10s-20s）

| 条目 | 内容 |
|------|------|
| **镜头 2a** | 中景（MS），固定镜头。男孩从书包拿出面包。 |
| **镜头 2b** | 中近景（MCS），固定镜头，过男孩肩看过去——男孩视线方向。观众第一次跟随男孩视线发现狗。 |
| **镜头 2c** | 中近景（MCS），固定镜头。从另一个方向看男孩——他看到了什么。 |

### 场景 3：试探与分享（20s-35s）

| 条目 | 内容 |
|------|------|
| **镜头 3a** | 中景（MS），固定镜头。男孩低头看狗，慢慢掰开面包。 |
| **镜头 3b** | 特写（CU），固定镜头。面包被掰成两半——细节记录，不煽情。 |
| **镜头 3c** | 中景（MS），固定镜头。男孩把一半面包放在地上。 |
| **镜头 3d** | 过肩（OTS），从男孩肩后看狗走近、犹豫、后退、再上前。 |
| **镜头 3e** | 中近景（MCS），固定镜头。狗低头嗅面包，开始吃。男孩嘴角的微小变化。 |

### 场景 4：靠近与连接（35s-50s）

| 条目 | 内容 |
|------|------|
| **镜头 4a** | 中景（MS），固定→缓推。狗吃完面包，没有离开。它看着男孩。 |
| **镜头 4b** | 中景（MS），固定镜头。男孩轻轻伸出手。 |
| **镜头 4c** | 中近景（MCS），极缓推镜。狗走近，坐下，犹豫——然后把头靠在男孩膝盖上。 |
| **镜头 4d** | 特写（CU），极缓推镜。男孩的手指轻轻落在狗头上，开始抚摸。 |

### 场景 5：陪伴（50s-60s）

| 条目 | 内容 |
|------|------|
| **镜头 5a** | 中景（MS），固定→极缓拉远。一人一狗并排而坐，男孩抚摸狗，书包和水瓶在旁边。 |
| **镜头 5b** | 远景（WS），极缓拉远+定镜。从更远处看他们——两个小身影在广阔的日落中。叙事结束。 |

## 六、光影与色彩系统

### 全片色彩脚本

阶段 1：「落日黄金」· 场景 1-2（0s-20s / 33%）

| 条目 | 内容 |
|------|------|
| **主调色板** | 金橙（#D4A373）、暖棕（#8B5E3C）、天空淡蓝（#A8C5D6） |
| **饱和度** | 中高——黄金时刻浓郁 |
| **反差** | 中——阳光强度仍较高 |
| **色温** | 暖（~5500K→4500K 夕阳推移） |
| **视觉参考** | 夏日傍晚的饱和暖光，影子开始拉长 |

阶段 2：「暖意交融」· 场景 3-4（20s-50s / 50%）

| 条目 | 内容 |
|------|------|
| **主调色板** | 暖橙（#E78F5E）、琥珀（#BC6C25）、柔金（#F4E4C1） |
| **饱和度** | 中——太阳渐低，光线变柔和 |
| **反差** | 中→低——阴影变长变柔 |
| **色温** | 暖（~4000K→3500K） |
| **视觉参考** | 日落前最柔和的时刻，光晕笼罩 |

阶段 3：「暮色余温」· 场景 5（50s-60s / 17%）

| 条目 | 内容 |
|------|------|
| **主调色板** | 暖紫（#B5838D）、暗蓝（#4A4E69）、最后一抹金（#E8C07A） |
| **饱和度** | 低中——光在消退 |
| **反差** | 低——光线均匀漫射 |
| **色温** | 中性转微凉（~3500K→5000K） |
| **视觉参考** | 日落后光线最柔和的蓝色时刻（Blue Hour） |

### 光影推进图

**场景 1（0s-10s）：黄金时刻主光**
- 主光：太阳（右侧上方 45°），暖金色硬光，色温 ~5000K
- 辅光：天空漫射光补阴影，柔
- 轮廓光：逆光勾勒男孩轮廓
- 阴影：长、硬（日落典型的拉长阴影）
- 调色板锁定：暖金主调，天空区域蓝色保留

**场景 2（10s-20s）：黄金时刻延续**
- 主光：同上，太阳角度略低，色温 ~4500K
- 辅光：地面反射光给阴影补暖
- 特殊：男孩脸部分在光中、部分在阴影中——光暗交界
- 调色板锁定：与场景 1 连续

**场景 3（20s-35s）：光开始柔化**
- 主光：太阳更低，色温 ~4000K，穿过空气层产生漫射
- 辅光：增加，阴影变柔
- 体积光：可能出现微弱光束（低角度太阳穿过扬尘/薄雾）
- 调色板锁定：暖橙为主，反差降低

**场景 4（35s-50s）：最暖时刻**
- 主光：太阳接近地平线，色温 ~3500K，低角度从侧面照亮
- 辅光：大量天空光漫射，阴影几乎消失
- 轮廓光：非常柔和的金色轮廓光从西侧来
- 特殊：所有表面都笼罩在暖色光晕中
- 调色板锁定：温暖包裹感——画面中最暖的段落

**场景 5（50s-60s）：蓝色时刻过渡**
- 主光：太阳已落下，残余天光从上方漫射
- 辅光：环境光均匀
- 特殊：远方地平线还有一丝暖光，前景开始变暗
- 剪影：角色可能形成剪影或半剪影
- 调色板锁定：暖紫→暗蓝渐变，最后一抹金色在地平线

## 七、视觉连续性规则

### 跨镜头保持的约束

1. **日落光线推进**：光从暖金 → 暖橙 → 紫蓝自然过渡，每个场景都比前一个场景暗 1-2 档。光线角度随时间降低。

2. **男孩位置一致**：在所有场景中，男孩坐在同一位置、同一朝向。书包和水瓶在镜头 1a 放置后保持不变的位置。

3. **狗的外观一致性**：小黄狗——黄色短毛，中等体型（约 40-50cm 肩高），瘦（流浪状态），眼神温和。

4. **空间地理**：场景发生在一个开阔的公共空间（河边台阶/公园长椅/社区空地），背景有树木和远山/建筑。空间在场景 1 用远景建立后不再改变。

5. **180 度规则**：保持轴线连续——男孩面朝画面右/东南方看日落，狗从左侧入画。

6. **情绪克制**：Observer 人格要求——不滥用特写，不过度强调感性瞬间。让画面自己说话。
