# 日落·男孩与流浪狗 — 电影级短片提示包

## 项目元数据

- **总时长**：15 秒
- **画幅比例**：16:9 横屏
- **视觉风格**：电影级写实，黄金时刻自然光
- **导演人格**：Emotional（情绪导演）— 摄影机与角色共情
- **目标平台**：本提示包为文本级设计文档，经确认后进入分镜蓝图生成阶段

---

## 一、叙事概览

### 故事梗概

黄昏落日，一个男孩独自坐在草地上，正准备吃手中的面包。一只流浪小黄狗从旁出现，怯怯靠近。男孩看了看面包，又看了看小狗——短暂的犹豫后，他将面包掰成两半，与小狗分享。小狗吃掉面包后，在男孩身边坐下，轻轻把头靠在他膝盖上。男孩微怔，随即露出微笑，伸手抚摸小狗。一人一狗并排坐在夕阳中，旁边放着书包和小水瓶。这不是故事的终点，而是陪伴的起点。

### 叙事结构（三幕）

**第一幕：孤独的黄昏（~0-3s）**
- 场景目的：建立环境基调 + 角色现状（孤独一人）
- 情绪：平静中带着落寞，强度 3/10

**转折点 1：** 小黄狗从画面左侧出现

**第二幕：相遇与分享（~3-10s）**
- 场景目的：通过分享行为建立信任与联结
- 情绪：好奇 → 犹豫 → 温柔 → 信任，强度从 5 升至 8
- 因果链：小狗出现 → 男孩发现 → 决定分享 → 掰面包 → 小狗接受 → 靠头

**转折点 2：** 小狗将头靠在男孩膝盖上（全片情绪峰值）

**第三幕：陪伴（~10-15s）**
- 场景目的：新状态呈现——从孤独到有人陪伴
- 情绪：温暖 → 安宁 → 完满，强度 7 → 6 → 5

### 情绪弧线

```
强度
8 ┤          ╭──小狗靠头（最治愈时刻）
7 ┤     ╭────╯  ──── 抚摸
6 ┤    ╱            ╲
5 ┤   ╱ 相遇          ╲──陪伴
4 ┤  ╱
3 ┤ ╱ 孤独
2 ┤╱
  └───────────────────────────
  0s    4s     8s    12s   15s
```

## 二、分镜设计方案

### Part 1：日落到陪伴（0-15s）

15 秒单个 Part，含 4 个连续镜头，自然过渡。

---

#### SHOT 01：孤独的黄昏

| 维度 | 内容 |
|------|------|
| **时长** | ~3s |
| **场景** | 开阔草地/山坡，日落方向在画面右后方。天空暖橙到玫瑰色渐变。 |
| **目的** | 建立——引入环境、角色、现状 |
| **视觉** | 男孩侧身坐在草地上，书包和水瓶放在他右侧。他低着头，手中拿着一个面包，逆光勾勒出轮廓。远处有树影或地平线。 |
| **动作** | 男孩安静坐着，低头看向手中的面包。他缓慢地开始撕开面包的包装纸。 |
| **摄影机** | 中全景 [Medium Wide Shot] [平视] [静态] [深焦] |
| **构图** | 三分法——男孩置于右侧 1/3 线，目光方向（左侧）留有负空间。夕阳在背景中。 |
| **灯光** | 主光：落日逆光（右后 45°，暖金 ~3800K）；辅光：天空漫反射（冷 ~6500K）；角色有金色边缘光轮廓；面部在阴影中产生冷蓝→暖橙过渡 |
| **情绪** | 孤独、平静但略带落寞（强度 3）→ 为相遇做铺垫 |
| **角色锁定** | 男孩引用 `@[图片1]`，服装/外观以此为准 |

#### SHOT 02：相遇与分享

| 维度 | 内容 |
|------|------|
| **时长** | ~5s |
| **场景** | 同上一镜头，但小狗从左侧入画。 |
| **目的** | 推进——相遇触发 → 分享决策 → 分享执行 |
| **视觉** | 小狗从画面左侧边缘出现，停止，看向男孩（和面包）。男孩察觉到它，抬头。他轮流看向面包和小狗——短暂的犹豫。然后他低下头，专注地将面包掰成两半。将其中一半放在地上推向小狗。小狗小心靠近，嗅了嗅，叼走了半块面包。 |
| **动作** | 小狗出现 → 站定，耳朵微竖 → 男孩抬头，视线来回切换 → 男孩掰面包（动作小心，面包碎屑落下）→ 放到地上 → 小狗靠近 → 叼走面包 |
| **摄影机** | 中近景 [Medium Close-Up] → 过肩（小狗视角看男孩）[平视] [缓推镜——情绪推进] [浅景深] |
| **调度** | 焦点拉焦转移：小狗（出现）→ 男孩（犹豫）→ 面包（掰开）→ 小狗（接受） |
| **构图** | 开始：男孩视线方向有空间让小狗入画。分享时：构图紧凑，聚焦面包和手。过肩镜头：从小狗肩膀后方看男孩，建立关系视角。 |
| **灯光** | 男孩身体微转，使面部从阴影转向侧光。掰面包时，手部被金色侧光照亮。小狗进入逆光区，边缘有暖色光晕。 |
| **情绪** | 好奇（5）→ 犹豫（6）→ 温柔（7）→ 信任建立（7） |
| **角色锁定** | 男孩 `@[图片1]` / 小黄狗 `@[图片2]` |

#### SHOT 03：联结（情绪峰值）

| 维度 | 内容 |
|------|------|
| **时长** | ~3s |
| **场景** | 同前，视角收窄到二人关系的最亲密处。 |
| **目的** | 强化——全片情绪的顶点：信任被确认的一刻 |
| **视觉** | 小狗吃完面包后在男孩身边坐下来。它短暂地看了看男孩，然后非常轻地、缓缓地，把头靠在了男孩的膝盖上。男孩的身体微微一僵——他被这个动作触动了。 |
| **动作** | 小狗坐下 → 侧头看向男孩 → 慢慢把头放下来靠在膝盖上 → 合上眼睛或半眯着眼。男孩微怔——上半身几乎静止，只有呼吸在加速。 |
| **摄影机** | 特写 [Close-Up] [平视] [静止——不切，不退] [极浅景深，焦点在面部的接触点] |
| **构图** | 负空间构图——狗的头和男孩的膝盖在画面中心偏下，周围环绕落日虚化的暖色光晕和草地。不挤压情绪——画面留出呼吸空间。 |
| **灯光** | 全片最暖时刻。金橙色逆光，接触点有微妙的高光。冷蓝色阴影已完全退去，转为暖紫色柔影。边缘光在狗耳朵和男孩的手指上闪烁。 |
| **情绪** | 感动、治愈、心被触动（强度 8）——这是整部影片的核心帧 |
| **角色锁定** | 男孩 `@[图片1]` / 小黄狗 `@[图片2]` |

#### SHOT 04：陪伴

| 维度 | 内容 |
|------|------|
| **时长** | ~4s |
| **场景** | 回到最初的取景范围，但构图从不对称变为平衡。 |
| **目的** | 收束——呈现新状态：从孤独到陪伴 |
| **视觉** | 男孩慢慢地、温柔地将手放在小狗的背上，轻轻地抚摸。小狗的尾巴在草地上微微摆动。一人一狗并排坐在落日中。夕阳继续沉降，光线变得更暖更柔和。影子拉得更长。书包和小水瓶安静地放在他们右侧。 |
| **动作** | 男孩微笑 → 伸手抚摸小狗背部（从脖子到背部，缓慢的来回）→ 小狗尾巴摆动 → 男孩放松肩膀 → 他们并排坐着，一起看向前方 |
| **摄影机** | 中全景 [Medium Shot] [平视] [极缓横移——如同确认般的轻微运动] [中景深→逐渐清晰的全景] |
| **构图** | 从不对称三分法过渡到**对称构图**——一人一狗并排居中。左侧书包+水瓶平衡画面。天空、地平线、二人剪影形成稳定三角形。 |
| **灯光** | 夕阳继续沉降，色温降至 ~3500K。金琥珀色光包裹二人。逆光剪影效果渐显，边缘发丝和毛发呈金色。阴影被拉长到画面之外。 |
| **情绪** | 温暖、安宁、完满（强度 6 → 5）——宁静的结束 |
| **角色锁定** | 男孩 `@[图片1]` / 小黄狗 `@[图片2]` |

## 三、摄影机语言规范

### 摄影机运动词汇

| 运动 | 使用镜头 | 叙事意图 |
|------|---------|---------|
| **静态** | SHOT 01, SHOT 03 | 孤独时让观众沉浸在空寂中；联结时让情绪有空间沉淀 |
| **缓推** | SHOT 02 | 情绪逐渐深入——从观察到参与的视觉过渡 |
| **拉焦转移** | SHOT 02 内部 | 引导注意力：物（面包）→ 人（表情）→ 动物（接受） |
| **极缓横移** | SHOT 04 | 摄影机"点头确认"——用微小运动肯定新状态 |

### 构图哲学

| 阶段 | 构图 | 含义 |
|------|------|------|
| 孤独期 | 三分法 + 负空间 | 画面"缺一半"，暗示缺失的陪伴 |
| 分享期 | 紧凑构图 | 关系拉近，世界缩小到二人之间 |
| 联结期 | 负空间/呼吸感 | 情绪太重，画面不挤压——留白给感受 |
| 陪伴期 | 居中对称 | 稳定、平衡、完满——画面不再"缺一半" |

### 景别分布

| 景别 | 占比 | 使用镜头 |
|------|------|---------|
|中全景（MS/WS） | ~47% | SHOT 01, SHOT 04 |
| 中近景（MCU） | ~33% | SHOT 02 |
| 过肩（OTS） | SHOT 02 内部 | 视角切换 |
| 特写（CU） | ~20% | SHOT 03 |

## 四、灯光与色彩脚本

### 灯光系统

- **主光策略**：落日自然逆光（右后 45°），全片唯一主光源
- **色温弧线**：从 4500K（偏冷黄昏）→ 3800K（暖金）→ 3500K（琥珀沉降）
- **反差规则**：孤独期 = 中反差（暖光/冷影对比）；联结期 = 低反差（全暖调）；陪伴期 = 柔和反差

### 色彩脚本

| 阶段 | 主色 | 色温 | 饱和度 | 视觉印象 |
|------|------|------|--------|---------|
| **孤独** | 暖橙 + 冷蓝灰 | 暖冷对比 | 中 | 金色逆光拖长影，远处冷蓝色 |
| **相遇** | 暖金 | 暖 | 中高 | 光渐暖，冷色消退 |
| **分享** | 琥珀金 | 暖 | 高 | 双手被暖光包裹 |
| **联结** | 深金 | 暖 | 高 | 无冷色，全暖调覆盖 |
| **陪伴** | 金琥珀 → 暖红 | 暖 | 中 | 光沉降，色彩沉淀 |

## 五、声音设计方向

### 音乐风格

- **风格**：极简温暖原声/钢琴独奏
- **节奏/BPM**：60-70 BPM，缓慢呼吸感
- **乐器**：钢琴（单音/分解和弦）为主，辅以极轻微弦乐铺底
- **乐句**：长持续音 + 稀疏音符——不填满空间

### 声音叙事

| 时间 | 声音 | 功能 |
|------|------|------|
| 0-3s | 环境音（微风、远处鸟鸣）+ 极缓钢琴单音 | 建立空旷、孤独的听觉空间 |
| 3-5s | 钢琴音渐入第二音，环境音中增小狗的轻微喘息声 | 新角色的听觉引入 |
| 5-7s | 面包掰开的清脆声被放大，钢琴音略升 | 分享的听觉锚点 |
| 7-10s | 环境音减弱，音乐渐强——温暖和弦进入 | 情绪从视觉转移到听觉 |
| 10-11s | **音乐与音效同时暂停 0.5s**——绝对静默 | 靠头时刻的情绪重量——听觉上的"瞬间凝结" |
| 11-14s | 音乐从暂停处缓缓恢复，更温暖的和弦 | 抚摸时的听觉温暖感 |
| 14-15s | 音乐渐弱至消失，留下非常微弱的黄昏风声 | 开放的结束——陪伴持续 |

### 静默运用

- **结构性静默**：靠头时刻（SHOT 03）——短暂的全静默，让观众"听见"这个瞬间的重量
- 这是全片唯一的声音断裂点，也是最重要的情绪锚点
