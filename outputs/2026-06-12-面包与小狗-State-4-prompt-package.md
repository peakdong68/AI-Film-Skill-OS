# 面包与小狗 (Bread & Puppy) — 电影级短片提示包

## 项目元数据

- **总时长**：15 秒
- **画幅比例**：16:9 横屏
- **视觉风格**：3D 动画，皮克斯级渲染质感，温暖治愈
- **导演人格**：Emotional Director（情绪导演）
- **目标平台**：AI 图像生成器（用于分镜蓝图生成）→ 视频生成

## 一、叙事概览

### 故事梗概

日落时分，放学路上的男孩拿出面包准备吃。一只流浪小黄狗坐在不远处，用悲伤而充满希望的眼神看着面包。男孩犹豫片刻，将面包掰成两块，碎屑在金色逆光中飘落。他把半块递向小狗。小狗试探后轻轻接走。它走近男孩，坐在他身边。男孩抚摸它，小狗把头靠在男孩膝盖上。夕阳下，一人一狗并排而坐，书包和水瓶静静陪伴。

### 叙事结构

```
ACT I — 饥饿与注视 (0s–5s)
  SHOT 01: 男孩取出面包，准备咬下
  SHOT 02: 小狗悲伤凝视面包
  → 建立困境：一方饥饿，一方拥有

ACT II — 分享的决定 (5s–10s)
  SHOT 03: 男孩掰开面包，碎屑落下
  SHOT 04: 男孩递出半块面包
  SHOT 05: 小狗上前接面包
  → 转折：善意战胜自私

ACT III — 陪伴的回馈 (10s–15s)
  SHOT 06: 小狗坐男孩身边，男孩抚摸它
  SHOT 07: 小狗头靠男孩膝盖
  SHOT 08: 一人一狗并排而坐，书包水瓶在旁
  → 收束：分享换来陪伴
```

## 二、分镜设计方案

### Part 1：面包与小狗（0s–15s）

#### SHOT 01 — 男孩准备咬面包（0s–2.5s）

- **时长**：2.5 秒
- **场景**：户外空地，日落时分。男孩独自一人，书包放在身旁地上。背景可见柔和夕阳天空和远处模糊的树木剪影。
- **目的**：建立情境——普通日常一幕，即将被打破
- **视觉**：男孩居中偏左（三分法），上半身中景。右手拿着一个圆形小面包举至嘴边，嘴微张准备咬下。背景为暖金色天空，浅景深虚化。
- **动作**：男孩站立/半坐着，举面包靠近嘴边，动作自然日常。微驼背（背书包习惯），呼吸平稳。面包距嘴约 5cm 时停顿。
- **摄影机**：中景 (MS) / 平视 / 定镜→微缓推 (Static → Slow Dolly-in) / 浅景深
- **灯光**：主光——落日从画面左上 45° 射入，色温 ~3500K 暖金。辅光——天空环境光柔化右侧阴影。轮廓光——逆光在男孩发丝和肩膀边缘形成金色光边。
- **情绪**：平静 (3/10) → 注意力集中 (4/10)
- **角色锁定**：男孩——圆脸深棕圆杏眼 / 深棕黑短发 / 白T恤深蓝短裤 / 军绿书包和透明水瓶在身旁地上

#### SHOT 02 — 小狗悲伤凝视（2.5s–4.5s）

- **时长**：2.0 秒
- **场景**：同一空地，小狗位置在男孩右侧约 2–3 米处。蹲坐在地面上。
- **目的**：引入冲突——小狗的渴望与资源的稀缺
- **视觉**：小狗近景 (CU)，居中偏右。小狗坐姿，头微仰，眼睛固定凝视面包方向。背景虚化——暖金色散景。
- **动作**：小狗完全静止。耳朵向两侧微展（悲伤耳位）。尾巴静止下垂环在爪旁。眼神不眨眼，从下方往上看。偶尔极轻微舔嘴。上眼睑微下垂（悲伤感）+ 大瞳孔（希望感）= 标志性"悲伤请求"表情。
- **摄影机**：近景 (CU) / 平视（小狗视线高度） / 定镜 / 浅景深
- **灯光**：主光——落日从左上射入，在狗眼形成大而明亮的琥珀色高光（关键）。辅光——天空光柔化右侧。轮廓光——逆光勾勒狗耳边缘和吻部毛发。
- **情绪**：悲悯/渴望 (5/10)
- **角色锁定**：小黄狗——楔形头立耳微折 / 深琥珀色圆眼大高光 / 暖土黄短毛奶油腹 / 左耳尖浅色斑 / 微瘦中型幼犬

#### SHOT 03 — 掰面包特写（4.5s–6.5s）

- **时长**：2.0 秒
- **场景**：男孩手部特写。双手握住面包，背景极虚化——暖金色模糊光斑。
- **目的**：转折时刻——决定性的分享动作
- **视觉**：超特写 (ECU)，居中构图。男孩双手握住面包。手指先轻捏面包边缘（犹豫），然后用力缓慢掰开。面包从中断裂，细小碎屑在逆光中飘落呈金色光点。
- **动作**：双手拇指各按面包一侧。停顿 0.5s（决定瞬间）。双手缓慢向两侧用力，面包裂开。碎屑从裂口飘落，逆光中缓坠。
- **摄影机**：超特写 (ECU) / 微俯 45° / 定镜 / 极浅景深
- **灯光**：主光——落日逆光从画面上方偏左射入，穿透碎屑产生体积光/丁达尔效应。碎屑呈半透明金色。
- **情绪**：内心挣扎 → 决断 (6/10)
- **角色锁定**：男孩手部——儿童手型，手指纤细圆润，指甲干净短小。白T恤袖口隐约可见。

#### SHOT 04 — 男孩递出面包（6.5s–8.5s）

- **时长**：2.0 秒
- **场景**：中近景，男孩半身。画面右侧边缘可见小狗的模糊轮廓。
- **目的**：善意表达——分享的邀请
- **视觉**：中近景 (MCU)，三分法构图——男孩在左 2/3，右手臂向右侧伸展，手中托着半块面包。面包在画面中央偏右为视觉焦点。小狗模糊轮廓在右侧边缘。
- **动作**：男孩右手缓缓从胸前向右前方伸出，掌心向上托面包。手臂运动缓慢稳定。目光从面包转向右侧小狗眼睛方向，嘴角微动，眼神柔化。身体微前倾。
- **摄影机**：中近景 (MCU) / 平视 / 缓推 (Slow Dolly-in) / 浅景深
- **灯光**：主光——落日从左上射入，面包被暖金光照亮。色温 ~3000K。
- **情绪**：善意/温柔 (6→7/10)
- **角色锁定**：男孩——面部完整出现。小黄狗——右侧边缘模糊轮廓。

#### SHOT 05 — 小狗上前接面包（8.5s–10.5s）

- **时长**：2.0 秒
- **场景**：近景，小狗为主体。男孩的手从画面左侧边缘伸入，托着面包。
- **目的**：接受善意——跨物种信任建立
- **视觉**：近景 (CU)，三分法构图。小狗在画面中右位置，从坐姿变为站立。男孩的手从左侧伸入，面包在画面中央。小狗头部向面包移动。
- **动作**：小狗看面包→看一眼男孩确认安全→前腿迈半步→停顿→再迈半步→轻轻叼走面包。尾巴在叼走瞬间第一次抬起慢摇。
- **摄影机**：近景 (CU) / 平视（小狗高度） / 定镜 / 浅景深
- **灯光**：主光——落日从左上射入，色温 ~2800K 浓金。
- **情绪**：试探→接受→感激 (7→8/10)
- **角色锁定**：小黄狗——完整近景。男孩——仅右手入画。

#### SHOT 06 — 男孩抚摸小狗（10.5s–13.0s）

- **时长**：2.5 秒
- **场景**：中景，男孩和小狗并排。小狗坐于男孩右侧，身体间距约 10cm。夕阳更低，天空暖金→琥珀。
- **目的**：情感升温——从施与受转为平等陪伴
- **视觉**：中景 (MS)，三分法构图。男孩在左 1/3，小狗在右 1/3。男孩右手放在小狗背上抚摸。小狗身体靠向男孩，尾巴慢摇。
- **动作**：男孩右手从肩胛向后背缓慢抚摸（完整循环约 1.5s）。肩部放松。小狗身体靠向男孩腿，耳朵从紧张变放松前倾，尾巴持续慢摇。
- **摄影机**：中景 (MS) / 平视 / 微环绕 (Slow Orbit ~15°) / 浅景深
- **灯光**：主光——落日更低位，色温 ~2800K 琥珀金。影子更长更柔。
- **情绪**：温暖/亲密 (8→9/10)
- **角色锁定**：男孩和小狗——并排关系，间距约 10cm。

#### SHOT 07 — 小狗头靠膝盖（13.0s–15.0s）

- **时长**：2.0 秒
- **场景**：超特写，膝盖高度。小狗头部与男孩膝盖的接触点。背景极虚化深琥珀色散景。
- **目的**：全片情感最高点——无声的信任与依赖
- **视觉**：超特写 (ECU)，居中构图。小狗头部侧面靠在男孩膝盖上。眼睛半闭，身体完全放松。男孩深蓝色短裤面料在小狗头下。男孩左手轻放狗背不动。
- **动作**：小狗缓慢将头侧向靠下——头部重量完全交付到男孩膝盖。眼睛睁开→缓慢半闭，慢速眨眼。尾巴慢摇→逐渐停止。男孩手静止不动。
- **摄影机**：超特写 (ECU) / 平视（膝盖高度） / 定镜 / 极浅景深
- **灯光**：主光——落日极低位，色温 ~2600K 深琥珀。光比极低，整体柔和统一。
- **情绪**：治愈/信任 (9→10/10) → 宁静回落 (10→7/10)
- **角色锁定**：小黄狗——头部侧面特写 + 半闭眼。男孩——膝盖和左手。

#### SHOT 08 — 并排远景（15.0s–16s 定格淡出）

- **时长**：1.0 秒定帧
- **场景**：远景，整个场景。男孩和小狗并排坐在画面右下。书包和水瓶在左侧。远处夕阳近地平线，天空琥珀→淡紫暮色。
- **目的**：收束余韵——空间包裹两人
- **视觉**：远景 (WS)，三分法构图。男孩和小狗在右下 1/3，夕阳在左上。温暖剪影感，带暖色轮廓光。
- **动作**：完全静止，定格 1s 后淡出。
- **摄影机**：远景 (WS) / 平视 / 定帧 / 深焦
- **灯光**：主光——落日 ~2600K 深琥珀。天空带暮色紫调微染。人物接近剪影带暖轮廓光。
- **情绪**：宁静/余韵 (7→5/10)
- **角色锁定**：男孩和小黄狗——远景轮廓，书包水瓶在旁。

## 三、摄影机语言规范

### 摄影机运动词汇

| 运动 | 叙事意图 | 使用镜头 |
|------|----------|----------|
| 定镜 (Static) | 给情绪沉淀时间 | SHOT 02, 03, 05, 07 |

| 微环绕 (Slow Orbit) | 温暖包裹，亲密感可视化 | SHOT 06 |

### 构图哲学

| 取景方式 | 使用时机 |
|----------|----------|
| 三分法 | 大多数叙事镜头——SHOT 01, 04, 05, 06, 08 |

| 负空间 | SHOT 02 小狗独坐孤独感，SHOT 08 天地开阔 |

| 景别 | 占比 | 叙事理由 |
|------|------|----------|

## 四、灯光与色彩脚本

### 灯光系统

- **主光策略**：落日（色温、高度、强度随时间演进）
- **色温弧线**：3500K → 3200K → 2800K → 2600K（全片逐步变暖变深）
- **反差规则**：中反差起 → 中反差 → 低反差（太阳越低光比越小）
- **光源方向**：画面左上 45° 射入，全片不跳轴

## 五、声音设计方向

### 音乐风格

- **风格**：温暖治愈的钢琴独奏 + 柔和弦乐铺底
- **BPM**：60–70（慢板，与情绪弧线同步）
- **调性**：大调为主，带轻微小调色彩（苦乐参半的温柔）
- **情绪推进**：
  - 0–5s：钢琴单音/简单琶音，稀疏留白
  - 5–6.5s（掰面包）：轻柔弦乐和弦进入
  - 6.5–10s（递+接）：钢琴旋律展开，弦乐渐厚
  - 10–13s（抚摸）：旋律温暖上行，弦乐最丰满
  - 13–15s（头靠）：旋律最高点后轻柔回落
  - 15s 结尾：钢琴延音自然衰减

### 声音叙事

- **环境音**：轻柔微风（不抢戏）、远处隐约鸟鸣（1–2 声）
- **音效重点**：
  - SHOT 03：面包掰开的轻微撕裂声（foley）
  - SHOT 03：碎屑飘落极轻微沙沙声
  - SHOT 07：几乎完全静默
- **静默运用**：SHOT 02 凝视时音乐退至几乎无声；SHOT 07 头靠时纯静默或极轻单音
