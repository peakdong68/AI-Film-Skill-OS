# STATE 3: 角色锁定 — 面包与小狗

## 角色 A：男孩（Boy）

### 1. 角色身份核心

- 叙事角色：主角 (Protagonist)
- 年龄范围：9–11 岁
- 性别表达：男孩，柔和中性
- 情绪身份：善良、安静、略带孤独感的学童
- 心理画像：内向温柔，放学路上独自一人。看到流浪小狗时本能地想分享而非驱赶。孤独让他理解另一个孤独的生命。
- 故事功能：善意/分享的发起者——从独食者转变为分享者
- 角色弧线：孤独放学 → 看见小狗犹豫 → 决定分享 → 获得陪伴

### 2. 视觉身份系统

**面部系统（3D 动画风格）：**

| 参数 | 锁定值 |
|------|--------|
| 脸型 | 柔和椭圆，颧骨轻微可见，婴儿肥残留 |
| 肤色 | 暖象牙色，微带日晒后的浅蜜色 |
| 眼型 | 圆杏眼，双眼皮，眼角微微下垂（温柔感） |
| 眼珠颜色 | 深棕色 (#4A2C1A)，高光点明显 |
| 眉毛 | 自然微粗，深棕色，柔和弧度 |
| 鼻子 | 小巧圆鼻头，鼻梁柔和低矮（儿童比例） |
| 嘴型 | 小嘴，上唇薄下唇略饱满 |
| 下颌/下巴 | 柔和圆润，儿童钝感，无棱角 |
| 辨识特征 | 左眉尾有一小颗浅痣 |
| 面部结构 | 儿童头身比，额头略大，五官集中于面部下半 |

**发型系统：**

| 参数 | 锁定值 |
|------|--------|
| 发型 | 自然短发，微碎层次，刘海自然垂落至眉毛上方 |
| 长度 | 刘海到眉上 0.5cm，两侧到耳垂 |
| 颜色 | 深棕黑 (#2A1A0A) |
| 质地 | 直发偏软，微蓬松 |
| 分线 | 自然无分线，随发旋自然散落 |
| 动态 | 微风拂动时有轻微飘动，整体保持形状 |

**体型系统：**

| 参数 | 锁定值 |
|------|--------|
| 体型 | 偏瘦型儿童身材 |
| 身高 | 约 130–140cm |
| 体态 | 偏瘦但不羸弱，肩宽自然 |
| 姿势 | 日常微驼背（背书包的习惯），放松时身体略微内收 |
| 动作签名 | 走路步伐偏慢，膝盖微内八，思考时喜欢低头 |

**服装系统：**

| 参数 | 锁定值 |
|------|--------|
| 标志服装 | 白色棉质短袖T恤 + 深蓝色棉质短裤 |
| 配色方案 | 白 (#F5F0E8) + 深蓝 (#1E3A5F) + 暖灰 (#C8C0B8) |
| 面料语言 | T恤：薄棉质，微透光；短裤：棉质斜纹，略厚 |
| 版型 | T恤略宽松落肩，短裤及膝松紧腰 |
| 配饰 | 白色运动袜，蓝色帆布鞋 |
| 叠穿 | 无叠穿（夏日单层） |

**标志道具：**

| 道具 | 详细描述 |
|------|----------|
| 书包 | 双肩背包，军绿色帆布材质，略有磨损，正面小口袋，拉链带橙色拉环。书包略鼓，内有书本。 |
| 小水瓶 | 透明塑料水瓶，瓶身有淡蓝色刻度线，挂于书包侧袋。瓶内剩半瓶水。 |

### 3. 镜头行为系统

**情绪→动作映射：**

| 情绪 | 身体表现 |
|------|----------|
| 日常平静 | 肩膀微塌，呼吸平稳，步态缓慢 |
| 注意（看见小狗） | 身体静止，重心微前倾，手停在空中 |
| 犹豫/内心挣扎 | 微低头看面包，手指轻捏面包边缘 |
| 决断（掰面包） | 双手用力，身体微前倾，下颌微收 |
| 善意（递面包） | 手臂缓慢伸出，掌心向上，身体前倾 |
| 喜悦/满足（抚摸） | 肩放松，微笑浅淡但抵达眼角，抚摸节奏平缓 |
| 被依赖（头靠膝盖） | 身体静止，呼吸放缓，手轻放在小狗背上不动 |

### 4. 连续性锁定

**硬锁（不可变）：** 面部、发型、体型、服装、年龄、道具
**软变化（允许）：** 表情递进、光影变化、景别角度差异、微风效果
**禁止变化：** 换脸、换装、发型突变、年龄跳变、痣位移

## 角色 B：小黄狗（Puppy）

### 1. 角色身份核心

- 叙事角色：配角 / 情感催化剂
- 年龄范围：约 6–8 个月（幼犬期末）
- 物种/品种：中华田园犬，黄色短毛
- 情绪身份：警觉渴望、流浪中保持希望、温柔天性
- 心理画像：孤独的流浪小狗，饥饿但仍有尊严。不莽撞抢夺，而是用悲伤而充满希望的眼神无声请求。
- 故事功能：接受善意 → 回馈信任，完成"分享→陪伴"的情感循环
- 角色弧线：流浪饥饿 → 无声请求 → 获得分享 → 靠近 → 交付信任

### 2. 视觉身份系统

**头部系统：**

| 参数 | 锁定值 |
|------|--------|
| 头型 | 楔形略宽，额段适中，幼犬特征（头略大于身体比例） |
| 耳型 | 立耳微折，三角形，耳尖圆钝，随情绪转动 |
| 鼻镜 | 黑色，湿润反光 |
| 吻部 | 中等长度，不尖不钝，幼犬钝感 |
| 嘴型 | 闭合时嘴线微微上翘（天生微笑感），微张嘴时露出粉色舌头边缘 |

**眼部系统（核心情感窗口）：**

| 参数 | 锁定值 |
|------|--------|
| 眼型 | 中等偏大圆眼，眼角微下垂（悲伤感），眼距适中 |
| 眼珠颜色 | 深琥珀色 (#8B6508)，湿润高光点明显且大——关键特征 |
| 眼周 | 深色眼线轮廓，让眼神更清晰 |
| 眼神特征 | 悲伤而充满希望——上眼睑微下垂产生悲伤感，大瞳孔+湿润高光产生希望感 |

**毛发系统：**

| 参数 | 锁定值 |
|------|--------|
| 毛色 | 主色：暖土黄色 (#C89850)；腹部/吻部下侧：浅奶油色 (#F0E0C0) |
| 毛质 | 短毛，贴身，微光泽 |
| 毛长 | 躯干短毛（~1cm），尾部稍长 |
| 换毛/斑点 | 无斑点，纯色渐变 |

**体型系统：**

| 参数 | 锁定值 |
|------|--------|
| 体型 | 中型犬幼犬，微瘦（可隐约看到肋骨轮廓但不夸张） |
| 身高/体长 | 肩高约 30–35cm，体长约 45–50cm |
| 体态 | 偏瘦但不羸弱，四肢比例协调 |
| 尾巴 | 中等长度，毛稍长，蓬松微卷。情绪关键指示器 |

**标志特征：**
- 左耳尖有一小块浅色毛（视觉锚点）
- 湿润大眼 + 大瞳孔 = 核心身份锚点
- 尾巴的状态变化是情绪的"实时仪表"

### 3. 镜头行为系统

**情绪→动作映射：**

| 情绪 | 身体表现 |
|------|----------|
| 饥饿/渴望 | 坐姿，头微仰盯面包，偶尔舔嘴，尾巴静止下垂 |
| 谨慎/戒备 | 身体重心后移，前腿微曲准备后退，耳朵后转 |
| 悲伤/请求 | 坐姿静止，头微低但从下往上看，大眼湿润，耳朵向两侧平展（悲伤耳位） |
| 犹豫/试探 | 前腿迈半步→停顿→再迈半步，重心低，尾巴低位慢摇 |
| 接受/感激 | 轻轻叼走面包，尾巴第一次真正抬起慢摇 |
| 信任/亲近 | 主动走近，身体靠向男孩腿，呼吸放缓 |
| 交付/依赖 | 整个头重量放在膝盖上，眼睛半闭，身体完全放松，尾巴缓慢摆动 |

### 4. 连续性锁定

**硬锁（不可变）：** 头型、眼型/颜色、毛色/毛质、体型、左耳标记
**软变化（允许）：** 耳位变化、尾巴状态、表情递进、光影反射
**禁止变化：** 品种突变、毛色变化、眼珠颜色变化、体型比例改变、标记消失

## 多角色关系系统

```
角色关系图：

男孩 ←────→ 小黄狗
(发起者)      (回应者)

关系弧线：陌生人 ──分享──→ 接受者 ──靠近──→ 陪伴者 ──依赖──→ 相互归属
权力动态：男孩持有资源（面包）→ 平等分享 → 小狗交付信任（主动靠头）
```
