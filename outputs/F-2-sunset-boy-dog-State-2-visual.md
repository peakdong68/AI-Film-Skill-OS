# STATE 2 — 视觉设计

## 项目：日落·男孩与流浪狗 (F-2)

---

## 导演人格：Emotional Director（情绪导演）

> 摄影机与角色共情。它参与到角色的情绪状态中。
> 特写主导、慢推、浅景深——捕捉微表情、呼吸、犹豫。

### 视觉哲学
情绪驱动每一个决策。摄影机不是观察者——它是情绪的延伸体。

### 摄影哲学
- **主导镜头类型**：Close-up（情绪细节）> Medium（互动叙事）> Wide（环境建立）
- **运动词汇**：Slow dolly-in（情绪深入）> Static（呼吸空间）> 极缓轨道（关系建立）
- **角度特征**：Eye-level 亲密角度为主，偶有轻微俯角（男孩视角看狗）
- **镜头性格**：浅景深主导，焦点在角色面部/互动点之间呼吸式游移

### 节奏哲学
- **节奏类型**：情感古典剪辑——情绪峰值时极致放慢
- **剪辑密度**：低（15 秒约 4-5 个镜头，平均 3-4s/镜）
- **静默运用**：结构性——小狗靠头的瞬间，画面"停下"给观众时间感受
- **情绪节奏**：慢 → 渐暖 → 最慢（情绪峰值）→ 缓释

### 光影哲学
- **光影理念**：Expressive naturalism——以自然落日为基础，强化其情绪表现力
- **主光质量**：柔和的黄金时刻暖光
- **色温弧线**：从偏冷黄昏 → 金色暖意渐浓 → 稳定暖琥珀

### 叙事哲学
- **叙事方式**：Show don't tell——用微表情、空间距离、光影变化叙事
- **信息策略**：情绪揭示——观众通过感受理解故事，而非通过情节

## 一、全片镜头语言系统

### 情绪→摄影机映射

| 情绪节拍 | 时间 | 镜头 | 运动 | 角度 | 焦点 | 构图 |
|---------|------|------|------|------|------|------|
| **孤独** | 0-2s | 中远景 | 定镜 | 平视 | 深焦 | 三分法，男孩置于右侧，左侧负空间 |
| **相遇** | 2-4s | 中景 | 缓推 | 平视 | 浅景深 | 男孩+小狗入画，空间压缩 |
| **犹豫** | 4-5s | 特写（面包）→ 特写（男孩表情） | 定镜 | 平视 | 拉焦转移 | 紧凑构图 |
| **分享** | 5-7s | 中近景（手部动作） | 缓推 | 轻微俯角 | 浅景深 | 聚焦掰面包的手 |
| **信任** | 7-8s | 过肩（从小狗背后看男孩） | 定镜 | 小狗平视 | 浅景深 | 过肩构图 |
| **联结** | 8-10s | 中近景（小狗） | 极缓推镜 | 小狗平视 | 浅景深 | 居中 |
| **治愈** | 10-11s | 特写（小狗靠头） | **静止/不切** | 平视 | 极浅景深 | 负空间围绕接触点 |
| **抚摸** | 11-13s | 超特写（手抚摸狗背）→ 缓拉至中近景 | 缓拉 | 平视 | 浅→中景深 | 展开构图 |
| **陪伴** | 13-15s | 中全景（一人一狗并排） | 极缓右移 | 平视 | 深焦 | 对称/平衡，书包+水瓶在左侧 |

### 运动语法总结

| 运动 | 使用场景 | 情绪功能 |
|------|---------|---------|
| **定镜** | 孤独时刻、治愈峰值 | 让观众沉入情绪，不打扰 |
| **缓推** | 相遇→分享·情绪上升期 | 拉近观众与角色的心理距离 |
| **缓拉** | 抚摸→陪伴·情绪释放期 | 从亲密感过渡到全景的陪伴感 |
| **拉焦转移** | 面包→男孩表情 | 引导注意力从物到人的情绪 |
| **极缓横移** | 最终并排 | 摄影机如同轻轻"点头"确认这个画面 |

### 景别序列规则

```
Wide（孤独建立）→ Medium（相遇互动）→ CU（情绪节点）
→ Med（行为叙事）→ OTS（视角切换）→ Med/CU（信任建立）
→ CU/ECU（情绪峰值——靠头时刻）
→ Med→Wide（情绪释放→陪伴状态）
```

### 连续性规则

- **180 度线**：以男孩面向的方向为轴线，摄影机始终在夕阳一侧
- **视线匹配**：男孩视线方向（先看面包→再看向小狗出现的方向→低头看狗）
- **空间地理**：男孩坐在画面偏右，小狗从左侧入画；书包和水瓶固定位于男孩右侧
- **景别递进**：不跳跃极端景别，每次过渡不超过两档

## 二、全片色彩脚本（单场景·情绪演进）

这是一个连续场景——色彩脚本在 15 秒内随光线自然演变。

### 情绪→色彩映射

| 情绪 | 主色 | 色温 | 饱和度 | 反差 | 视觉印象 |
|------|------|------|--------|------|---------|
| **孤独** | 暖橙+冷蓝灰 | 冷暖对比 | 中饱和度 | 中反差 | 金色逆光，长影，蓝色阴影区 |
| **相遇/好奇** | 暖金 | 暖 | 中高 | 柔和 | 光线渐暖，阴影收缩 |
| **分享/温柔** | 琥珀金 | 暖 | 高 | 低 | 暖光包裹双手和面包 |
| **联结/治愈** | 深金+柔琥珀 | 暖 | 高 | 低 | 全暖调，无冷色干扰 |
| **陪伴/安宁** | 金琥珀+暖红 | 暖 | 中 | 柔和 | 光线沉降，色彩趋暖暗 |

### 色彩演变曲线（15 秒）

```
暖 ┤              ╭───╮
   │         ╭────╯   ╰───╮
温 ┤    ╭───╯             ╰────
暖 ┤╭───╯
度 ┤│
冷 ┤╰─────────────────────────
   │
   0s────3s────5s────8s────11s────15s
   偏冷  暖金   琥珀   深金   金琥珀
  [孤独] [相遇] [分享] [联结] [陪伴]
```

### 场景调色板锁定

| 色彩元素 | 锁定内容 | 原因 |
|---------|---------|------|
| 暖色主调 | 全片以落日暖色为主导色温 | 叙事需要温暖感，冷色仅用于前期孤独对比 |
| 金色逆光边缘光 | 男孩和小狗均有金色边缘光 | 增强温暖感，分离主体与背景 |
| 蓝色阴影 | 前段保留蓝色冷阴影 | 强化孤独感，与暖光形成张力 |
| 阴影温度 | 随情绪推进，阴影从冷蓝渐变为暖紫 | 反映情绪从孤独到陪伴的转变 |

## 三、光影设计系统

### 主光源设定

| 参数 | 内容 |
|------|------|
| **主光源** | 落日（低角度逆光，位于地平线附近） |
| **光质** | 柔和的黄金时刻暖光（低角度散射+大气粒子） |
| **色温** | 3500K→4500K（随时间自然降低，色温略暖化） |
| **高度** | 低——约 10-15° 地平线，拉出长影 |
| **方向** | 逆光（从角色背后打来）+ 偏侧（右后方 45°） |

### 灯光推进图（单场景演进）

| 时间 | 灯光状态 | 变化动机 | 情绪功能 |
|------|---------|---------|---------|
| 0-2s | 全落日逆光，长影，冷蓝阴影区 | 落日自然状态 | 空寂、孤独感，对比强烈 |
| 2-5s | 光影不变，但小狗进入光区，产生第二组影子 | 新角色引入改变画面权重 | 注意力转移至新元素 |
| 5-8s | 男孩身体微转，面部从侧阴影转向迎光 | 分享动作让男孩面向光源 | 情绪升温，表情细节可见 |
| 8-11s | 小狗靠头，两者进入同一光区，边缘光融合 | 物理接触 = 光中融合 | 视觉上的"联结"具象化 |
| 11-13s | 抚摸动作，手划过光区，拉出暖色边缘光轨迹 | 动作激活光线 | 手的运动在光中留下温暖轨迹 |
| 13-15s | 光线略沉降，轮廓更柔和，阴影拉更长 | 时间的自然流逝 | 安宁、完满感 |

### 灯光设置详述

- **主光**：落日逆光（右后 45°），色温 ~3800K，柔光散射
- **辅光**：天空漫反射补光（冷色 ~6500K，弱强度），确保阴影不纯黑
- **轮廓/边缘光**：自然落日边缘光——男孩头发、肩膀轮廓；小狗耳朵、背部毛发光晕
- **眼神光**：落日方向——角色眼中暖金色反光点
- **阴影**：前段冷蓝（6500K 天空反射），后段暖紫（4000K 地面反射）
- **环境光**：天空环境光 + 地面反射暖光

### 光影连续性规则

- **主光方向始终一致**：右后 45°，不得跳跃至正面光或顶光
- **阴影方向不变**：所有阴影拉向左前方
- **色温变化只能变暖**：从 4500K → 3500K，不可回冷
- **边缘光不间断**：角色边缘落日轮廓光在所有镜头中保持

## 四、构图规则

### 主导取景方式

| 规则 | 应用 | 场景 |
|------|------|------|
| **三分法（前期）** | 男孩置于右侧 1/3 线，左侧负空间 | 孤独建立——空荡感 |
| **对称（后期）** | 一人一狗居中并排 | 陪伴——平衡、稳定、完满 |
| **负空间利用** | 男孩方向前留空，暗示小狗出现方向 | 引导观众视线预备小狗入画 |
| **框中框** | 可选——利用前景草地/植物框架 | 增加偷窥/见证感 |

### 深度分层设计

| 层 | 内容 | 功能 |
|----|------|------|
| **前景** | 虚化草地/地面纹理 | 增加景深感，呼吸空间 |
| **中景** | 男孩、小狗（主动作区域） | 叙事主体 |
| **背景** | 落日、天空、远山/树影 | 建立环境，情绪定调 |

男孩与小狗的距离演变：**物理距离 = 情感距离**
- 孤独期：距离远（心理孤岛）
- 相遇期：距离缩小
- 分享期：相邻
- 联结期：接触（靠头）
- 陪伴期：并排

## 五、运镜分节拍（15 秒·4 个连续镜头建议）

### 镜头 1：孤独的黄昏（0-3s）

```
Shot type: MS/WS（中全景）
Angle: Eye-level
Movement: Static
Lens: Deep focus, 暖金+冷蓝对比
Framing: 三分法——男孩右侧，左侧负空间
Duration: ~3s
```

### 镜头 2：相遇与分享（3-8s）

```
Shot type: MCU（中近景）→ 过肩切换
Angle: Eye-level, 偶有轻微俯角
Movement: Slow dolly-in（情绪推近）
Lens: Shallow DOF, 拉焦（面包→男孩→小狗）
Framing: 紧凑构图，视线引导
Duration: ~5s
```

### 镜头 3：联结（8-11s）—— 情绪峰值

```
Shot type: CU（特写——狗靠头瞬间）
Angle: Eye-level
Movement: Static（停在此时刻）
Lens: 极浅景深，焦点在接触点
Framing: 负空间围绕头靠膝盖的接触点
Duration: ~3s（尽可能停留，给观众时间感受）
```

### 镜头 4：陪伴（11-15s）

```
Shot type: MS（中景——一人一狗全景）
Angle: Eye-level, 缓拉至二人关系
Movement: 极缓横移（如同点头确认）
Lens: 中景深，渐清晰的全景
Framing: 从三分法过渡到居中对称构图
Duration: ~4s
```
