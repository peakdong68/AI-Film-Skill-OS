# STATE 1 — 故事结构与情绪弧线设计

## 项目：日落·男孩与流浪狗 (F-2)

---

## 一、叙事结构图（三幕结构 — 15s）

```
第一幕：孤独的黄昏（~0-3s）
  ─ 男孩独自坐于落日中，拿出面包。孤独、平静、略带落寞。
  
  转折点 1：一只小黄狗从画面边缘出现，怯怯接近。

第二幕：分享与信任（~3-10s）
  ─ 男孩注意到小狗→低头看面包→看向小狗→决意分享。
  ─ 他将面包掰成两半，将一半递给小狗。
  ─ 小狗迟疑→靠近→叼走面包→在男孩身边坐下。

  中点转折：小狗吃下第一口面包，信任建立。
  转折点 2：小狗轻轻把头靠在男孩膝盖上。

第三幕：新的陪伴（~10-15s）
  ─ 男孩微怔→露出微笑→伸手抚摸小狗的背。
  ─ 一人一狗并排而坐，夕阳将影子拉长。
  ─ 书包和水瓶静静放在一旁。
  ─ 新的平衡：从孤独到陪伴。
```

**因果链：**
```
男孩独自吃面包（孤独）→ 小狗出现（触发）→ 男孩掰面包分享（决策）
→ 小狗接受食物（信任）→ 小狗靠头（联结）→ 男孩抚摸（回应）
→ 并排而坐（新状态：陪伴）
```

## 五、情绪→视觉映射

| 情绪 | 摄影机 | 灯光 | 色彩 | 节奏 |
|------|--------|------|------|------|
| **孤独** | 远景/中景，固定镜头 | 金色逆光，剪影感 | 暖橙+冷蓝对比 | 缓慢，呼吸感 |
| **好奇** | 中近景，小幅横移 | 侧光，纹理显现 | 橙黄加深 | 略加速 |
| **犹豫** | 特写（男孩表情/面包） | 暖光为主 | 金色 | 停顿感 |
| **温柔/分享** | 过肩镜头，小狗视角 | 柔光 | 暖色渐浓 | 缓 |
| **信任/联结** | 近景，浅景深 | 日落黄金时刻暖光 | 全暖色调 | 最慢，停留在此刻 |
| **陪伴** | 中景，一人一狗同框 | 逆光剪影+边缘光 | 暖橙+金色 | 缓降，自然结束 |

## 六、节奏结构

```
0s ───── 2s ───── 5s ───── 8s ───── 11s ───── 15s
慢        中慢      慢        最慢      缓降
[孤独]    [相遇]   [分享]    [联结]    [陪伴]

剪切节奏：
- 0-3s：长镜头，无切（建立氛围）
- 3-8s：缓切，每个动作为一个镜头（掰面包→递出→小狗吃）
- 8-11s：停在此刻，不切（小狗靠头——情绪峰值，给观众时间感受）
- 11-15s：缓切至结束（抚摸→并排→夕阳定格）

全片约 4-5 个镜头，自然过渡。
```
