# STATE 2: 视觉设计 — 面包与小狗

## 一、导演人格：Emotional Director（情绪导演）

**视觉哲学**：摄影机与男孩和小狗一同呼吸——它不是旁观者，而是另一个温柔的存在。每一个镜头选择都服务于情绪共振。

## 二、摄影机语言系统

**主导镜头语法：**

| 类型 | 叙事分配 |
|------|----------|
| 特写 (CU) / 超特写 (ECU) | 情绪核心——小狗眼神、掰面包的手指、头靠膝盖 |
| 中近景 (MCU) | 角色关系——男孩递面包、小狗接面包 |
| 中景 (MS) | 叙事动作——男孩取面包、抚摸小狗、并排而坐 |

**运动词汇（Emotional profile）：**

| 运动 | 使用场景 | 情绪功能 |
|------|----------|----------|
| 定镜 (Static) | 小狗凝视、掰面包、头靠膝盖 | 给情绪沉淀时间，让观众"看进去" |
| 缓推 (Slow Dolly-in) | 男孩取面包、递面包 | 逐步进入角色内心 |
| 缓拉 (Slow Dolly-out) | 结尾并排远景 | 温柔抽离，留余韵 |

**焦点策略**：全片浅景深为主——背景虚化突出主体，焦距在男孩与小狗间自然转移。

**取景规则**：
- 三分法：大多数镜头的默认取景
- 居中构图：掰面包特写（仪式感）、头靠膝盖（情感锚点）
- 负空间：小狗独坐时的孤独感

| # | 景别 | 角度 | 运动 | 构图 | 情绪意图 |

## 四、灯光设计系统

**全片光源设定：日落黄金时刻**

```
主光：落日（画面左侧/上方偏左）
- 色温：~3200K→2600K（随太阳下沉逐渐变暖变深）
- 质感：柔和方向光，通过大气散射自然柔化
- 方向：从左上 45° 射入

辅光：天空环境光
- 色温：~4000K（暖中性天空散射）
- 强度：主光的 30-40%
- 用途：柔化阴影，保持暗部细节

轮廓光：落日逆光
- 色温：~2800K（金色逆光）
- 用途：分离角色与背景，赋予边缘温暖的高光
```

## 五、色彩脚本（3D 动画色板）

```
ACT I — 饥饿与注视 (0s–5s)
  主色：暖金琥珀 (#E8A850)、柔粉 (#D4A090)
  饱和度：中高
  反差：中低
  色温：暖 (3500K)
  氛围：温暖的日常感中带一丝忧郁——阴影略偏暖灰

ACT II — 分享的决定 (5s–10s)
  主色：浓金 (#D49040)、奶白 (#F5E6D0)、暖褐 (#8B5E3C)
  饱和度：高
  反差：中
  色温：暖 (3200K→2800K)
  氛围：落日最浓烈的时刻——面包碎屑在逆光中闪耀，金色包裹一切

ACT III — 陪伴的回馈 (10s–15s)
  主色：琥珀深金 (#C87830)、暖橙 (#E89860)、暮紫 (#C0A0B0)
  饱和度：中高 → 逐渐柔和
  反差：中低
  色温：暖 (2800K→2600K)  渐入暮色
  氛围：温暖深沉——太阳更低，影子更长，光线更柔和，暮色带紫调微染
```
