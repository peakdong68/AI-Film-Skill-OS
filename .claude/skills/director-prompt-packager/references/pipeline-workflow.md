# Seedance 制作管线工作流 — 内置参考知识

为 `director-prompt-packager` 提供 Seedance 制作管线知识。

## 专业制作流程

1. **Brief（简报）：** 定义客户/创意目标、受众、区域、时长、画幅比例、交付物、参考素材、权利、审批负责人和硬约束。
2. **Pre-production（前期制作）：** 创建设计方案、参考映射、分镜列表、连续性账本、色彩/音频/本地化意图和风险日志。
3. **Generation plan（生成计划）：** 拆分为稳定片段。每个 Seedance 片段分配一个可见节拍、一个摄影机构思和一个终点。
4. **Review loop（审核循环）：** 在扩展或编辑之前评估身份、产品、动作、摄影机、连续性、音频同步、文字、安全性和权利。
5. **Post plan（后期计划）：** 编辑、合成、稳定化、声音、色彩、字幕、版本管理、无字版和归档元数据。
6. **Delivery/QC（交付/质控）：** 检查规格、命名、帧率、分辨率、色彩管线、响度、字幕、安全区域、权利备注和人工审核。

## 工作流拆分

| 工作流 | 说明 |
|--------|------|
| Web 工作流 | 即梦/可灵前端，上传参考，输入提示词，审核输出 |
| API 工作流 | Volcengine/BytePlus/Runway，模型 ID，认证，文件处理，任务生命周期 |
| 专业制作工作流 | 设计方案、分镜列表、连续性账本、参考权利映射、审核循环、后期交接 |
| 后期工作流 | 编辑、合成、拼接、稳定化、音频清理、字幕、色彩、本地化、版本管理 |
| 首/末帧工作流 | 映射首帧、末帧、转场动作、身份锁定、终点目标 |
| 扩展工作流 | 从现有最终状态继续，使用最后一帧作为连续性锚点 |

## 连续性规则

- 每个 Part ≤ 15 秒（Seedance 单次生成上限）。
- Part 1 建立世界基线：角色身份、服装、环境、灯光逻辑、摄影机语言。
- Part 2+ 必须引用前一段视频输出作为连续性锚点。
- 前向传递：角色身份、服装、环境、灯光逻辑、摄影机语言。
- 情绪演进而非重置——从上一 Part 的终点继续。

## 多 Part 连续性模板

```
Part 1: 建立世界基线 + 角色 + 视觉语言
Part 2: 引用 Part 1 视频 → 延续同一角色/服装/空间/灯光/摄影机 → 情绪从终点继续
Part N: 引用 Part N-1 视频 → 叙事推进 → 收束
```
