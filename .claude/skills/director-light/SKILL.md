# Director Light — 色彩与灯光智能引擎

## 概览

设计赋予电影情绪质感的视觉情绪系统。此技能构建两个互连系统：一个色彩脚本，将情绪旅程在全时长上映射为特定调色板；一个灯光设计，为每个场景指定光源、质感和推进。输出确保每个帧的视觉一致性和情绪意图性。

可独立用于灯光/色彩设计，也可被 `director-core` 在 STATE 2 调用。

## 加载资源

此技能附带参考知识文件。需要时加载共享资源 `../references/cinematography-quick-reference.md` 获取灯光速查表。

## 核心原则

> 光与色是情绪演员，不是环境默认值。

每个灯光和色彩决策必须服务于叙事。暖色、冷色、反差、饱和度——这些都是控制观众感受的工具，而不仅是审美偏好。

## 输出结构

### 1. 全片色彩脚本

设计跨越全片时长的色彩旅程——类似于皮克斯色彩脚本。对每个叙事阶段（来自 `director-emotion` 的情绪弧线）：

```
阶段 1 [名称]：[时长占比%]
- 主调色板：[2-3 种颜色，附 hex 或描述名]
- 饱和度水平：[低 / 中 / 高]
- 反差水平：[低 / 中 / 高]
- 色温：[暖 / 冷 / 中性 / 混合]
- 视觉参考：[一行外观描述]

阶段 2 [名称]：[时长占比%]
...
```

色彩脚本必须显示推进——色彩必须随故事演进，不能保持静态。

### 2. 情绪→色彩映射

定义色彩语言：

| 情绪 | 主色 | 色温 | 饱和度 | 反差 |
|------|------|------|------|------|
| 悲伤/孤独 | 蓝灰 | 冷 | 低饱和 | 低 |
| 爱/浪漫 | 暖金、琥珀 | 暖 | 中 | 柔和 |
| 恐惧/恐怖 | 绿、暗低饱和 | 冷 | 极低 | 高 |
| 愤怒/冲突 | 红、高反差 | 热 | 高 | 极高 |
| 平静/解决 | 柔蓝、暖白 | 中性暖 | 中 | 低 |
| 神秘 | 深紫、阴影 | 冷暖混合 | 可变 | 高 |
| 力量/奢华 | 黑、金、银 | 冷 | 低中 | 高 |
| 希望/乐观 | 暖黄、柔绿 | 暖 | 中 | 柔和 |

### 3. 光影推进图

定义灯光如何跨越叙事推进：

```
场景 1：[灯光设置] → 情绪功能：[为什么]
场景 2：[灯光设置] → 如何从场景 1 演变
...
```

对每个场景的灯光进行指定：

**灯光类型：**

| 类型 | 情绪功能 |
|------|---------|
| 自然光 | 写实感、纪实感 |
| 硬光 | 冲突、压力、残酷真相 |
| 柔光漫射 | 浪漫、梦境、回忆 |
| 霓虹/彩色光 | 科技、夜晚、都市、超现实 |
| 剪影/逆光 | 神秘、情绪隐藏、揭示 |
| 实景光 | 叙事内光源（灯、窗、屏幕） |
| 体积光 | 氛围感、光束、丁达尔效应 |
| 闪烁光 | 不稳定、张力、恐惧 |

**灯光方向：**
- 主光位置和动机
- 辅光强度与质感
- 轮廓光/逆光分离主体
- 每个光源的色温
- 阴影质感（硬/柔）和方向

### 4. 场景专属调色板

对每个场景锁定调色板：

```
场景[N]：[名称]
- 主光：[来源、方向、色温、质感]
- 辅光：[来源、强度、色温]
- 轮廓光：[来源、颜色、用途]
- 环境光：[整体色调偏移]
- 阴影：[密度、色彩偏移]
- 特殊：[体积光、实景光、反射]
- 调色板锁定：[此场景中必须保持一致的元素]
```

### 5. 光温情绪效果

| 光影特征 | 情绪效果 |
|----------|---------|
| 暖光 | 亲密、安全、怀旧 |
| 冷光 | 孤立、恐惧、理性 |
| 混合色温 | 情绪冲突、过渡状态 |
| 闪烁光 | 不稳定、紧张、危险 |

**色彩脚本规则：**
- 色彩必须随叙事演变——静态调色板 = 错失机会
- 每个叙事阶段需要有定义的调色板
- 色温变化必须有动机（时间、地点、情绪）
- Pixar 风格方法：每个情绪节拍对应一个主导色

### 6. 冷暖变化曲线

追踪全时长上的色温变化：

```
暖 ┤         ╭──╮
   │        ╱    ╲
中性 ┤───────╱      ╲──────
   │      ╱        ╲
冷 ┤╭────╱          ╲────
   │╲
   0%──────25%──────50%──────75%──────100%
```

以色温推移标注叙事事件——从暖到冷的转变应对应一个故事节拍。

### 7. 光影连续性

必须跨全片保持的规则：

- 每场景的主光方向必须保持一致
- 色温变化必须有动机（时间变化、地点变化、情绪转变）
- 阴影质感应演进，不能任意重置
- 在一个镜头中建立的实景光源必须在同一场景的后续镜头中持续存在
- 不存在无动机的"美颜光"——每个光源必须有叙事或叙事内原因

## 情绪灯光模板

常用灯光情绪的速查：

**浪漫：**
- 一侧暖色主光（黄金时刻或实景灯）
- 柔光辅光减弱阴影
- 微妙暖色轮廓发光
- 浅景深配散景
- 轻微暖色调调色

**张力/惊悚：**
- 非常规角度硬主光（低角度或高角度）
- 深阴影配合最小辅光
- 冷色温
- 实景光源（闪烁荧光灯、路灯）
- 高反差调色

**奢华/商业：**
- 大柔光主光均匀照明
- 强力轮廓光/逆光定义产品边缘
- 冷调阴影增加深度
- 表面受控反射
- 干净高调调色

## 约束

- 每个场景必须有定义的主光光源——不存在纯环境光照明。
- 色彩必须随叙事演进——静态调色板 = 错失机会。
- 灯光连续性必须在分镜交接前验证。
- 同一场景的镜头间不存在无动机的色调跳变。

## 集成

被 `director-core` 调用时：
- 加载 `director-emotion` 的情绪蓝图构建情绪→色彩映射
- 与 `director-camera` 对齐确保统一的视觉语言
- 场景调色板系统直接输入分镜和提示词编译
