# Director Interview — 创意入站访谈

## 概览

统一入站访谈逻辑。根据用户输入的创意清晰度，选择最佳路径将输入转化为可执行的制作简报。此技能被 `director-core`（STATE 0）和 `director-story`（独立使用）调用。

## 三路径判定

| 输入质量 | 判定标准 | 路径 | 输出 |
|---|---|---|---|
| 创意完整 | 已含主体+动作+场景+情绪方向，可直接推进 | **快速通道** | 确认即过的制作简报 |
| 描述充分但缺剧情 | 有场景/风格/调性等描述，缺具体叙事动作和镜头节奏（不限品牌/商业，含主题氛围/风景纪录/抽象概念等） | **创意展开** | 2-3 方案 → 用户选择 → 简报 |
| 创意模糊 | 仅有主题词/宽泛概念/无具体场景和动作 | **创意访谈** | 3 问 → 回答 → 简报 |

## 快速通道

用户已提供主体+动作+场景+情绪方向，可直接推进。确认关键参数后输出简报：

- **时长、画幅、风格、平台** — 逐项确认
- **参考素材** — 主动询问："你手头是否有角色、场景或产品的参考图片/视频/音频可以提供？" 无论用户输入多完整，此项为必问。结果记入简报 `已有参考` 字段，供后续阶段使用。

用户确认后输出简报，无需额外提问。

## 创意展开（描述充分但缺剧情）

当用户提供了场景、风格、调性等描述性信息，但缺乏具体的叙事动作和镜头节奏，不要直接输出结构。先提供 2-3 个创意方案供选择：

```
根据您的素材，以下是几个创意方向：

方案A：[名称] — [一句话描述 + 适合时长 + 节奏特点]
方案B：[名称] — [一句话描述 + 适合时长 + 节奏特点]
方案C：[名称] — [一句话描述 + 适合时长 + 节奏特点]

您倾向哪个方向？或者有其他想法？
```

方案应覆盖不同风格（纯展示/微剧情/氛围MV/纪实/抽象表演等）。用户确认方向后，收集制作参数（时长、画幅、风格、平台、参考素材）并输出简报。其中**参考素材**为必问项——主动询问是否有角色/场景/产品参考图或视频/音频可提供。

## 创意访谈（模糊输入）

最多 3 个高影响力问题，只问缺失的关键信息：

1. **主体 + 变化**："谁是主角？Ta 在做什么？结束时发生了什么变化？"
2. **感觉 + 类型**："希望呈现什么感觉？产品展示感、剧情张力、喜剧轻松、写实、动画、还是氛围感？"
3. **参考素材**："有没有参考图/视频/音频？或者脑海中参考的影视风格？"

根据回答构建创意简报，再收集制作参数。

## 输出：制作简报

无论哪条路径，最终输出一份统一的制作简报：

```
## 制作简报

- 项目：[经确认的创意描述]
- 创意方向：[选择的方案或自然形成的方向]
- 类型路径：[产品展示/微剧情/氛围MV/纪实/叙事短片/...]
- 预期时长：[15s / 30s / 60s / 自定义]
- 视觉风格：[电影级 / 商业 / 纪录片 / 动漫 / 科幻等]
- 交付平台：[Seedance / Kling]
- 画幅比例：[16:9 / 9:16 / 1:1]
- 已有参考：[角色图/场景图/视频片段/音频片段/无]
```

## 原则

- 不问不需要的信息——用户已回答的维度不再重复提问
- 不输出结构——此阶段产出简报，非剧本或分镜
- 用户说"直接生成"时——用合理默认值填补并标注
