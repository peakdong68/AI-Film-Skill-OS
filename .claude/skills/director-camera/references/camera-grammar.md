# Camera Grammar Knowledge

## Shot Type Grammar

| Type | Narrative Function | Emotional Association |
|------|-------------------|----------------------|
| Wide Shot | Establish world, isolation, spatial relationships | Loneliness, scale, freedom |
| Medium Shot | Carry narrative action, behavior, interaction | Neutral, observational |
| Close-Up | Reveal emotion, internal state | Intimacy, intensity, vulnerability |
| Extreme CU | Decisive moment, emotional peak | Shock, revelation, anchor |
| Over-Shoulder | Relationship tension, dialogue | Conflict, connection, perspective |
| POV | Immersive experience | Identification, subjectivity |
| Low Angle | Power, threat, heroism | Dominance, intimidation, awe |
| High Angle | Vulnerability, overview | Weakness, surveillance, overview |
| Dutch Angle | Psychological instability | Unease, disorientation, madness |

## Camera Movement — Emotional Functions

| Movement | Emotional Function |
|----------|-------------------|
| Static | Oppression, observation, stillness, waiting |
| Slow Dolly-In | Emotional deepening, psychological pressure |
| Dolly-Out | Emotional withdrawal, loneliness, revelation |
| Tracking | Participation, pursuit, following action |
| Orbit | Complexity, relationship, romance, surrealism |
| Handheld | Realism, chaos, tension, documentary feel |
| Crane | Epic scale, spatial revelation, grandeur |
| Whip Pan | Emotional burst, rapid transition, shock |

## Emotion → Camera Mapping

| Emotion | Shot | Movement | Angle |
|---------|------|----------|-------|
| Sadness/Loneliness | Wide | Static / slow push-in | Eye level |
| Fear/Tension | Close-up | Handheld micro-shake | Dutch |
| Romance/Intimacy | CU / OTS | Slow orbit / dolly | Eye level |
| Mystery | Wide / silhouette | Static / slow reveal | Low |
| Power/Dominance | Low angle | Slow controlled dolly | Low angle |
| Shock | ECU | Sudden stillness | Eye level |
| Action/Urgency | Medium | Tracking | Dynamic |

## Composition Rules

- Rule of Thirds: Balanced, natural — default narrative
- Center Framing: Ritual, power, symmetry — significant moments
- Negative Space: Loneliness, isolation — subject pushed to edge
- Foreground Obstruction: Voyeurism, mystery
- Frame Within Frame: Layers, confinement, focus

## 180-Degree Rule

- Camera stays on one side of the action axis within a scene
- Crossing the line requires a neutral transition shot
- Screen direction (L→R / R→L) must not flip without explicit crossing

## Camera Prompt Template (per shot)

Shot type + Angle + Movement + Lens behavior + Framing + Emotional intent

## Negative Cinematography Rules

- No random camera shake
- No unnatural zoom jumps
- No inconsistent framing
- No broken spatial logic
- No floating objects without grounding
- No face distortion due to camera motion
