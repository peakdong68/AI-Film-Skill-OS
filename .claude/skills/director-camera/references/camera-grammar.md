# Camera Grammar Knowledge — 摄影机语法知识

## Shot Type Grammar（镜头类型语法）

| Type | 叙事功能 | 情绪联想 |
|------|---------|---------|
| Wide Shot | 建立世界、表现孤立、定义空间关系 | 孤独、尺度、自由 |
| Medium Shot | 承载叙事动作、表现行为与互动 | 中性、观察性 |
| Close-Up | 揭示情绪、内在状态 | 亲密、强度、脆弱 |
| Extreme CU | 决定性瞬间、情绪峰值 | 震惊、揭示、锚点 |
| Over-Shoulder | 关系张力、对话 | 冲突、连接、视角 |
| POV | 沉浸式体验 | 认同、主观性 |
| Low Angle | 力量、威胁、英雄主义 | 支配、威慑、敬畏 |
| High Angle | 脆弱、俯瞰 | 弱小、监视、全局 |
| Dutch Angle | 心理不稳定 | 不安、迷失、疯狂 |

## Camera Movement — Emotional Functions（摄影机运动——情绪功能）

| Movement | 情绪功能 |
|----------|---------|
| Static | 压迫、观察、静止、等待 |
| Slow Dolly-In | 情绪加深、心理压迫 |
| Dolly-Out | 情绪抽离、孤独、揭示 |
| Tracking | 参与感、追逐、跟随动作 |
| Orbit | 复杂性、关系、浪漫、超现实 |
| Handheld | 真实感、混乱、紧张、纪录风格 |
| Crane | 史诗尺度、空间揭示、宏大感 |
| Whip Pan | 情绪爆发、快速转场、震惊 |

## Emotion → Camera Mapping（情绪→摄影机映射）

| 情绪 | 镜头 | 运动 | 角度 |
|------|------|------|------|
| 悲伤/孤独 | Wide | Static / slow push-in | Eye level |
| 恐惧/紧张 | Close-up | Handheld micro-shake | Dutch |
| 浪漫/亲密 | CU / OTS | Slow orbit / dolly | Eye level |
| 神秘 | Wide / silhouette | Static / slow reveal | Low |
| 力量/支配 | Low angle | Slow controlled dolly | Low angle |
| 震惊 | ECU | Sudden stillness | Eye level |
| 动作/紧迫 | Medium | Tracking | Dynamic |

## Composition Rules（构图规则）

- Rule of Thirds（三分法）：均衡、自然——叙事默认
- Center Framing（中心构图）：仪式、权力、对称——重要时刻
- Negative Space（负空间）：孤独、孤立——主体推至画面边缘
- Foreground Obstruction（前景遮挡）：偷窥感、神秘感
- Frame Within Frame（框中框）：层次、局限、聚焦

## 180-Degree Rule（180度规则）

- 同场景内摄影机保持在动作轴同一侧
- 越轴需要中性过渡镜头
- 屏幕方向（左→右 / 右→左）不得无理由翻转

## Camera Prompt Template — per shot（单镜头摄影机提示词模板）

Shot type + Angle + Movement + Lens behavior + Framing + Emotional intent

## Negative Cinematography Rules（禁用摄影规则）

- 禁止随机画面抖动
- 禁止不自然的变焦跳跃
- 禁止构图不一致
- 禁止空间逻辑断裂
- 禁止无支撑的悬浮物体
- 禁止因摄影机运动导致的人脸变形
