# Seedance 摄影机镜头语言 — 内置参考知识

为 `director-camera` 技能提供 Seedance 平台特定的摄影机指导。

## 镜头合约（Shot Contract）

每个专业镜头应声明：

| 字段 | 决策内容 |
|------|---------|
| 景别（Shot size） | 超远景、远景、中景、近景、中特写、特写、超特写、微距 |
| 角度（Angle） | 平视、仰拍、俯拍、鸟瞰、侧面、过肩、插入、产品3/4角度 |
| 镜头感（Lens feel） | 广角空间感、自然35mm透视、人像压缩、微距材质细节 |
| 摄影机支撑 | 锁定、手持、滑轨、轨道、摇臂、无人机、稳定器、虚拟产品台 |
| 运动（Movement） | 推近、拉远、横移、环绕、摇镜、俯仰、升降、揭示、变焦 |
| 主体关系 | 摄影机跟随、引导、发现、保持、观察、遮挡主体 |
| 起始帧 | 第一个可读构图 |
| 结束帧 | 变化后的状态、姿势、揭示、产品特写、或连续性交接 |
| 脆弱锚点 | 面部、手部、logo、文字、产品形状、服装、道具位置 |

**Prompt 模式：**

`镜头：[景别/角度/镜头感]。摄影机起始[构图]，[一个运动]以[速度]运动，同时[主体动作]，结束于[明确终点]。保留[脆弱锚点]。`

## 景别使用指南

| 景别 | 用途 | Seedance 注意事项 |
|------|------|-----------------|
| 超远景 | 尺度、抵达、环境 | 小面部/logo会漂移；不要求面部表演 |
| 远景 | 调度、舞蹈、运动、环境中的产品 | 保持动作简洁可读 |
| 中景 | 对话、产品使用、交接 | 角色广告的良好默认 |
| 特写 | 情绪、纹理、唇形同步、产品细节 | 保持摄影机稳定、动作小 |
| 微距 | 材质、食物、珠宝、机械 | 避免大运动和文字重绘 |

## 摄影机运动语法

- **锁定（Locked-off）：** 用于唇形同步、产品身份、文字、logo、精确VFX、连续性锚点。
- **推近（Dolly/Push-in）：** 用于发现、领悟、亲密感、产品揭示。
- **横移（Lateral track）：** 用于旅行、行列、编舞、经过前景层。
- **环绕（Orbit）：** 用于产品英雄镜头和雕塑式主体；如果身份必须仅从单一角度保持稳定则避免。
- **摇臂/无人机（Crane/Drone）：** 用于尺度、抵达、地理、揭示；避免用于对话或微小产品文字。
- **手持（Handheld）：** 用于真实感或张力；当身份重要时保持运动微妙。
- **变焦（Rack focus）：** 用于两个锚定对象之间的注意力转移；避免与复杂摄影机运动叠加。

## 分镜与覆盖

多镜头场景保持摄影机语法有动机：

1. 建立地理和屏幕方向。
2. 移动到角色/产品动作。
3. 仅在细节改变故事时使用近景特写。
4. 每个片段以可作为下一段首帧参考的帧结束。

**覆盖模式：**

`镜头1 远景建立[空间/屏幕方向]。镜头2 中景跟随[动作]。镜头3 特写揭示[结果]。连续性：相同服装、道具、光线方向、视线。`

## 专业避用清单

避免无物理镜头选择的词汇：`电影感摄影机`、`动态镜头`、`史诗变焦`、`电影质感`、`好莱坞风格`、`美丽构图`。

替换为：景别、支撑方式、运动方式、主体关系、有动机的光线、终点、脆弱锚点。
