# Shot Semantics — 镜头语义系统

## 核心概念

每个镜头都有一个语义意图——不仅是展示了什么，还包含它在叙事中*意味着*什么。在选择镜头类型之前，导演必须先确定镜头的语义功能。

## 六种镜头语义

### 1. Observe（客观观察）
摄影机是中性目击者。不进行情绪操控。

**适用场景**：建立场景、信息传递、纪录性时刻
**摄影机行为**：Static、Eye-level、Medium-wide，不侵入画面
**情绪距离**：最大——观众保持客观视角

### 2. Immerse（沉浸体验）
摄影机将观众置于体验之中。

**适用场景**：动作段落、主观时刻、第一人称发现
**摄影机行为**：POV、Tracking、Handheld，流畅连续运动
**情绪距离**：零——观众就是参与者

### 3. Reveal（信息揭示）
摄影机控制观众知道什么、何时知道。

**适用场景**：情节转折、角色发现、解谜、产品亮相
**摄影机行为**：Slow push-in、Rack focus、遮挡→清晰、Crane reveal
**情绪距离**：被操控——导演控制信息流

### 4. Confront（情绪冲突）
摄影机强化角色之间或观念之间的情绪对抗。

**适用场景**：争吵、情绪对决、内心危机、道德抉择
**摄影机行为**：Tight framing、Close-up、Dutch angle、Handheld instability
**情绪距离**：极小——观众直接感受到张力

### 5. Isolate（孤立人物）
摄影机强调角色与世界的分离感。

**适用场景**：孤独、疏离、悲伤、决策时刻、权力关系
**摄影机行为**：Wide shot + Negative space、远距离构图、Shallow DOF 隔离
**情绪距离**：角色与世界之间的空间成为主体

### 6. Expand（空间扩展）
摄影机揭示世界的尺度，将个体渺小化。

**适用场景**：世界构建、敬畏感、渺小感、自由、旅程起点
**摄影机行为**：Extreme wide、Crane up、Drone pull-back、尺度的缓慢揭示
**情绪距离**：世界成为主体；角色成为语境

## 视觉优先级决策

在选择镜头语义之前，先判断此刻最重要的是什么：

| 优先级 | 摄影机聚焦于 | 对应语义 |
|--------|-------------|---------|
| 情绪优先 | 面部、肢体语言、微表情 | Immerse / Confront |
| 动作优先 | 身体运动、空间轨迹 | Immerse / Observe |
| 信息优先 | 物体、文字、环境细节 | Reveal / Observe |
| 关系优先 | 角色间距、调度关系 | Observe / Confront / Isolate |
| 尺度优先 | 环境、世界、建筑 | Expand / Isolate |

## Shot Semantics → Camera Mapping（语义→摄影机映射）

| 语义 | 镜头类型 | 运动 | 角度 | DOF |
|------|---------|------|------|-----|
| Observe | Wide / Medium | Static / slow track | Eye-level | Deep |
| Immerse | POV / Close-up | Tracking / handheld | Subjective | Shallow |
| Reveal | Medium → Close-up | Dolly-in / rack focus | Eye → detail | Shallow → Deep |
| Confront | Close-up / ECU | Handheld / tight static | Eye / Dutch | Shallow |
| Isolate | Wide / Medium | Static / slow pull-out | Eye / high | Deep or shallow |
| Expand | Extreme wide | Crane / drone / dolly-out | High / bird's eye | Deep |

## 语义递进（跨序列）

镜头语义应该递进演化，而非随机切换：

```
开场段落： Expand → Observe（先建立世界，再引入角色）
紧张推进： Observe → Confront（中性观察 → 情绪压迫）
发现时刻： Observe → Reveal → Immerse（看到 → 发现 → 感受到）
情绪高潮： Confront → Isolate（张力峰值 → 情绪余波）
收束段落： Isolate → Observe → Expand（角色 → 语境 → 世界）
```
