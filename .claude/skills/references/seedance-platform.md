# Seedance 2.0 平台约束 — 共享参考知识

## 提示词字数限制

| 语言 | 上限 | 理由 |
|------|------|------|
| **中文** | ≤ 500 字 | 字数过多导致信息分散，模型忽略细节 |
| **英文** | ≤ 1000 词 | 同上 |
| **总字符数** | ≤ 2000 字符 | seedance-prompt 平台预算 |

**违反此约束将导致视频缺失元素、角色漂移、动作不完整。** 编译完成后必须统计字数并标注。

## @[ref] 引用格式

所有上传的图像/视频/音频资产使用 `@[描述]` 格式引用：

| 引用示例 | 用途 |
|---------|------|
| `@[分镜图片1]` | 分镜蓝图——动作规划参考 |
| `@[角色图片1]` | 角色身份锁定——面部、发型、体型 |
| `@[产品图片1]` | 产品锁定——颜色、印花、版型 |
| `@[背景图片1]` | 环境锁定——空间、光线、色调 |
| `@[音频1]` | 节奏和能量——不复制受保护的声音/歌曲 |

## 稳定约束

- 不同 Seedance 2.0 表面的功能可能不同，不假设完全一致。
- 不根据记忆假设 API 访问、定价、模型 ID、区域限制、上传限额、时长或肖像授权。
- 不从未经授权的上传图像/语音/视频推断许可。
- 不提供受保护角色、名人、品牌 logo、歌曲复制、精确场景或声音模仿指令。
- 每 Part 视频 ≤ 15 秒（Seedance 单次生成上限）。
- Part 2+ 必须使用前一段视频输出作为连续性锚点。

## 表面特定约束

当用户提到即梦、可灵、Volcengine Ark、Runway 等特定平台时，回答中标注平台名称和日期。社区/非官方工具需明确标注。
