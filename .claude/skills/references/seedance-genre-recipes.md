# Seedance 类型配方 — 内置参考知识

标准化的视频类型模式与提示词骨架。作为起始模式使用，非刚性模板。选择匹配目标的配方，然后自定义主体、动作、摄影机、灯光、音频和约束。

## 配方家族

| 类型 | 最佳用途 | 核心模式 |
|---|---|---|
| 产品（Product） | 广告、电商、英雄镜头、材质揭示 | `产品锚点 + 一次材质变化 + 受控摄影机 + logo 保留` |
| 生活方式（Lifestyle） | 人物使用、美食、旅行、社交片段 | `简单动作 + 生活化环境 + 手持或自然光 + 环境音` |
| 剧情（Drama） | 情绪、对白、短叙事节拍 | `角色标签 + 动作 + 有动机的摄影机 + 静默或稀疏音效` |
| 音乐视频（Music video） | 节奏同步、舞蹈、风格化剪辑 | `节奏参考 + 可见节拍变化 + 光线脉冲 + 清晰角色调度` |
| 风景（Landscape） | 定场镜头、自然、氛围 | `慢速摄影机 + 天气运动 + 层次深度 + 自然音效` |
| 商业广告（Commercial） | 品牌安全的精致感和功能 | `问题/使用/结果节拍 + 精确产品约束 + 干净光线` |
| 动画（Animation） | 原创角色和风格化运动 | `媒介 + 形状语言 + 调色板 + 弹性或加权运动` |
| 视觉特效（VFX） | 变换、粒子、天气、能量 | `源 + 材质行为 + 交互 + 消散终点` |
| 首/末帧（First/last frame） | 中间过渡、产品状态变化、角色姿势目标 | `首帧 + 末帧 + 连续过渡 + 身份锁定` |
| 商业营销（Commercial campaign） | 6/10/15/30s 变体、竖屏/社交版、无字/本地化母版 | `钩子 + 产品证明 + 结束状态 + 版本矩阵 + 交付备注` |

## 提示词骨架

**产品 I2V：** `[Image1] 为产品参考；精确保留徽标、标签、形状及材质。[一项材质或灯光变化]。摄像机：[单一运动]。灯光：[物理光源]。音效：[环境音/音效]。`

**戏剧 T2V：** `角色 A 在 [具体场景] 中执行 [可见的情感动作]。摄像机：[有动机的构图]。灯光：[有动机的光源]。音效：[环境音或简短对话]。结束状态：[表情/动作变化]。`

**参考动作：** `[Video1] 仅提供 [摄像机/动作/时机] 参考；不迁移身份、服装、徽标或环境。新主体：[授权/原创主体]。[动作与终点]。`

**首/尾帧：** `[Image1] 为起始帧。[Image2] 为结束帧。保留 [身份/产品/场景锚点]。生成从 [起始状态] 到 [结束状态] 的连续过渡。摄像机：[锁定或单一可控运动]。音效：[环境音/音效]。`

**动画：** `原创 [角色原型] 在 [环境] 中执行 [动作]。风格：[媒介、线条质感、纹理、调色板]。动作：[节奏]。摄像机与音效：[简洁辅助]。`


## 选择规则

多目标竞争时，优先选择保护最脆弱需求的配方：

产品身份 > 摄影机奇观。
唇形同步 > 大幅度头部运动。
角色一致性 > 复杂编舞。
首/末帧目标精度 > 额外风格变化。
安全性和授权 > 风格模仿。
