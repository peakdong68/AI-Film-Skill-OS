# Anti-Slop Lexicon — Shared Reference Knowledge

Shared by all prompt-generation skills.

## Core Rule

Replace empty evaluation language with observable production language.

**If a camera, microphone, light meter, or stopwatch cannot detect it — rewrite it.**

## Replacement Table

| Weak Phrase | Replace With |
|---|---|
| cinematic / 电影感 | shot scale + camera movement + lighting + color grade |
| epic / 史诗 | physical scale + stakes + lens distance |
| beautiful / 美丽 | color + texture + composition + material + light behavior |
| stunning / 惊艳 | visible contrast + reveal + movement + detail |
| dynamic / 动态 | specific movement + speed + endpoint |
| dramatic / 戏剧化 | blocking + shadow + silence + camera pressure |
| ultra-realistic / 超写实 | material behavior + skin texture + lens artifacts + natural motion |
| cool transition / 酷炫转场 | match cut, whip pan, dissolve, hard cut, object wipe |
| magical / 魔幻 | particle behavior + glow source + motion path + interaction |
| premium / 高级 | material expression + light-shadow layering + composition approach |
| texture / 质感 | visible fabric fibers + light flowing across material surface |
| atmospheric / 氛围感 | light source color temperature + depth of field + environmental dynamics |
| professional / 专业 | product lighting setup + clean background + controlled camera |
