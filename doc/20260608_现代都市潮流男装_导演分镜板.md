# Storyboard Plan: 现代都市潮流男装 · 高品质电商直播

### Format: Fashion Director Board | 服装导演板

---

### Replaceable Zone 主体替换区

```
【场景】真实暖色衣帽间 — 木质衣柜、柔和暖光灯带、落地镜、男装展示间 — 干净白墙、挂杆陈列、自然光+暖色补光、温暖松弛氛围
【模特】年轻男性，25-30岁，清爽短发，都市潮流气质，自然淡妆，身高178-183cm，偏瘦匀称身形
【产品】现代都市潮流男装 — 廓形西装外套、落肩T恤、宽松直筒裤、机能风夹克、针织开衫、垂感阔腿裤；水洗质感、挺括版型、面料肌理清晰
【镜头动作】自然走入 / 整理衣摆 / 微转身展示廓形 / 单手插袋 / 对镜整理 / 坐姿松弛 / 行走展示动态垂感 / 回眸定格
```

### Bottom Module 1: Rhythm Structure 节奏结构（30秒 / 卡点剪辑）

```
| 0-5s    | 铺垫 & 空间建立 | 环境音+轻鼓点入场   | 镜头①②        |
| 5-10s   | 第一套look亮相   | 节奏律动推进        | 镜头③④⑤       |
| 10-17s  | 动态展示 & 细节   | 旋律推进+鼓点卡拍   | 镜头⑥⑦⑧       |
| 17-25s  | 生活方式松弛     | 节奏放缓+氛围铺陈    | 镜头⑨⑩⑪       |
| 25-30s  | 收尾强化品牌     | Drop/氛围音收束      | 镜头⑫          |
```

### Bottom Module 3: Lighting & Atmosphere 光影与氛围

```
✓ 左前方暖色主光（2700K-3200K 暖白）
✓ 顶部柔光箱辅助补光
✓ 右后方轮廓光（分离人物与背景）
✓ 衣帽间内置暖光灯带（实景光）
✓ 展示间自然光+暖色柔光混合
✓ 浅景深虚化背景（f/2.8-f/4 质感）
✓ 墙面干净通透、浅米色/暖灰色
```

**关键词:** 都市 / 松弛 / 高级感 / 质感 / 暖调 / 生活方式
