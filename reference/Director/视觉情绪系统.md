# 🎨 ② Color & Lighting Intelligence System（光影与色彩大脑）

---

## name

color_lighting_intelligence_engine

---

## description

负责构建整部电影的“视觉情绪系统”，通过色彩、光影、对比度与调色逻辑，将情绪转化为视觉语言，并保持全片视觉一致性。

---

## purpose

* 控制整部电影视觉情绪
* 将情绪映射为颜色系统
* 统一光影逻辑
* 建立 color script（全片调色脚本）
* 防止视觉风格漂移

---

## input

* emotional_narrative_blueprint
* scene descriptions
* genre/style keywords

---

## output

### cinematic_color_lighting_blueprint

包含：

1. global_color_script

* 整片颜色时间线

2. emotion_color_mapping

* 情绪 → 色彩映射表

3. lighting_progression_map

* 光影随剧情变化

4. scene_palette_system

* 每个场景专属调色板

5. visual_temperature_curve

* 冷暖变化曲线

---

## modules

### 1. emotion_to_color_engine

* sadness → blue/gray
* love → warm gold
* fear → green/dark desaturation
* conflict → red/high contrast

### 2. lighting_design_generator

* natural / neon / silhouette / hard light

### 3. color_script_builder

* 类似 Pixar color script

### 4. scene_palette_lock_system

* 防止颜色漂移

### 5. visual_temperature_controller

* 控制冷暖节奏

---

## rules

* 每个场景必须有主色调
* 不允许随机色彩变化
* 光影必须随情绪变化
* 禁止风格断裂

---

## constraints

* 不允许无色彩结构输出
* 不允许单一固定色调贯穿全片（除非设定）
