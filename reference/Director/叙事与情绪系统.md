
# 🎬 ① Emotion / Narrative Engine（叙事与情绪系统）

---

## name

emotion_narrative_engine

---

## description

将故事输入转化为“情绪驱动的电影结构系统”，负责生成情绪曲线、叙事节奏、剧情转折点，并将抽象情感映射为可执行的镜头与场景强度变化。该系统是AI Film OS的“情绪大脑”。

---

## purpose

* 将“剧情”转换为“情绪时间线”
* 控制观众情绪起伏
* 定义电影叙事节奏结构
* 绑定情绪 → 镜头 → 场景强度
* 驱动整个导演系统的节奏逻辑

---

## input

* story idea
* script
* scene description
* character context
* thematic keywords

---

## output

### emotional_narrative_blueprint

包含：

1. emotional_arc_map

* calm → tension → conflict → collapse → resolution

2. narrative_structure

* 3-act / 5-act breakdown

3. emotional_beats

* key emotional turning points

4. intensity_timeline

* 每个时间段情绪强度（1–10）

5. character_emotion_drift

* 每个角色情绪变化曲线

---

## modules

### 1. emotional_arc_generator

生成完整情绪曲线

### 2. narrative_turning_point_detector

识别剧情转折点

### 3. emotional_intensity_scorer

定义每个场景情绪强度

### 4. emotion_to_scene_mapper

情绪 → 场景结构转换

### 5. character_emotional_dynamics_engine

角色情绪变化模拟

---

## rules

* 所有故事必须具备情绪曲线
* 禁止“无情绪节点”的场景
* 每个场景必须绑定情绪值
* 情绪必须随时间变化，不可静态

---

## constraints

* 不允许纯剧情输出
* 不允许无情绪强度标注
* 不允许缺少转折点结构
