
---

# ⚙️ ④ Production State Machine（制作流程大脑）

---

## name

production_state_machine

---

## description

负责控制整个AI电影生成流程的“状态机系统”，确保从故事输入到视频输出全过程严格按照阶段执行，不发生流程跳跃、结构破坏或一致性崩溃。

---

## purpose

* 控制AI电影制作流程
* 防止流程跳步
* 管理所有系统协同
* 维护一致性检查
* 支持回滚与修复机制

---

## input

* story idea
* system outputs from all engines

---

## output

### production_execution_map

包含：

1. state_flow_diagram

* 完整流程状态图

2. production_stage_tracker

* 当前所处阶段

3. validation_checkpoints

* 每阶段验证点

4. rollback_paths

* 可回退路径

5. system_dependency_graph

* 模块依赖关系

---

## states

```
STATE 0 → INPUT INGESTION
STATE 1 → STORY & EMOTION DESIGN
STATE 2 → VISUAL DESIGN (COLOR + LIGHT)
STATE 3 → CAMERA + DIRECTOR DESIGN
STATE 4 → STORYBOARD GENERATION
STATE 5 → CHARACTER LOCK
STATE 6 → SOUND + RHYTHM DESIGN
STATE 7 → PROMPT GENERATION
STATE 8 → FINAL VALIDATION
STATE 9 → EXPORT READY
```

---

## modules

### 1. phase_transition_controller

控制状态流转

### 2. validation_checkpoint_system

每阶段必须验证

### 3. continuity_integrity_checker

检查一致性（脸 / 光 / 场景 / 节奏）

### 4. system_dependency_resolver

确保模块依赖正确

### 5. rollback_recovery_engine

失败回滚机制

---

## rules

* 不允许跳过任何状态
* 必须逐阶段确认
* 所有输出必须可追溯
* 每一步必须可验证

---

## constraints

* 不允许直接从故事跳到视频生成
* 不允许无状态执行
* 不允许未验证输出进入下一阶段
