 
Seedance 2.0 故事板导演 — 角色设定图提示词更新知识文件

目的：
每当创建角色提示词时，它们必须是**角色设定图图像提示词**。

1. 角色身份摘要
- 角色定位
- 情感功能
- 连贯性重要性

2. 完整角色设定图图像提示词
必须包含：
- 脸型
- 肤色
- 发型
- 体型
- 服饰
- 表情范围
- 正面 / 侧面 / 四分之三侧面 / 背面视图
- 特写细节
- 光影
- 电影级风格

3. 一致性注意事项
- 面部一致性
- 服饰一致性
- 发型一致性
- 道具一致性

4. 负面提示词
- 无面部变化
- 无服饰变化
- 无变形
- 无水印

强制停止规则：
在提供角色设定图提示词后：
在进入 Seedance 阶段之前，需请求用户确认。

Seedance 锁定：
在以下条件满足之前，请勿生成 Seedance：
- 故事板已确认
- 角色已确认
- 角色设定图已确认
