我给你完整拆解成 **“导演思维 + Prompt工程 + 规则系统”三层结构**，并给你一套可以直接复用的“生成公式”。

# 一、核心答案（一句话）

Seedance 2.0 的本质转换逻辑是：

> **把“分镜脚本”拆解为：时间 → 画面 → 摄影机 → 表演 → 光影 → 声音 → 动作连续性 → 再压缩成 AI 可执行的视频运动指令**

也就是：

**Storyboard（叙事） → Cinematic Grammar（电影语言） → Motion Prompt（机器语言）**

---

# 二、它依赖的7大核心技能系统

## 1. 🎬 电影导演语言（Cinematic Grammar）

来自知识文件中的核心能力：

* 镜头类型（wide / close-up / POV）
* 摄影机运动（dolly / tracking / orbit）
* 构图逻辑（中心对称 / 景深 / 前后景）
* 情绪控制（压迫 / 浪漫 / 疑问 / 恐惧）

👉 本质：
把“文字剧情”变成“可拍摄镜头”

---

## 2. 🧠 分镜拆解能力（Storyboard Deconstruction）

来自规则：

* 1个 storyboard = 8–12 panels
* 每 panel = 1个动作单元（1–3秒）
* 每个动作必须回答：

| 要素      | 问题     |
| ------- | ------ |
| Action  | 发生了什么？ |
| Emotion | 感觉是什么？ |
| Camera  | 怎么拍？   |
| Timing  | 几秒？    |

👉 本质：
把“故事”切成“时间碎片”

---

## 3. ⏱ 时间压缩系统（Temporal Mapping）

规则来源：

* 15秒 = 5–10 scenes
* 每 scene = 1.5–3秒

👉 转换逻辑：

```
Storyboard（静态结构）
→ Scene timing（时间流）
→ Motion segmentation（动作拆帧）
```

---

## 4. 🎥 摄影机语言系统（Camera Grammar Engine）

来自知识库 Camera Library：

必须明确：

* shot size（wide/close-up）
* angle（low/high/Dutch）
* movement（push-in/orbit/tracking）
* lens feel（macro, shallow DOF）

👉 这是 Seedance Prompt 的“骨架”

---

## 5. 🎭 表演行为转换（Actor-to-Motion Translation）

来自 Character Direction Library：

不是写：

❌ “她很伤心”

而是：

✔ “她低头，肩膀下沉，停顿呼吸，慢慢避开视线”

👉 AI video必须靠“动作行为”驱动，而不是情绪词

---

## 6. 🌫 环境与光影系统（Lighting-Mood Engine）

来自 Mood Library：

例如：

* romantic → warm gold + soft rim light
* horror → flickering cold shadows
* luxury → blue-silver reflective surfaces

👉 用于控制画面“质感”

---

## 7. 🔁 连续性锁定系统（Continuity Engine）

Seedance 关键能力：

* 同一角色必须保持：

  * face
  * wardrobe
  * lighting logic
  * environment

👉 所以 Prompt 必须包含：

* “same character identity”
* “continuous motion”
* “no change in wardrobe/face”
