如果要做到真正专业的故事板生成，仅仅有：

```text
场景
主体
动作
```

还不够。

影视行业的分镜设计实际上遵循的是：

```text
场景（Scene）
+
主体（Subject）
+
动作（Action）
+
镜头（Camera）
+
构图（Composition）
+
光线（Lighting）
+
情绪（Mood）
+
叙事目的（Story Purpose）
```

# 进一步升级：场景生成模板

当你还没有具体场景时，可以让AI先生成场景描述。

例如：

```text
根据以下主题生成专业故事板场景描述：

主题：
未来赛博朋克城市

输出内容：

1. 场景描述
2. 主体描述
3. 动作描述
4. 镜头建议
5. 光线建议
6. 情绪建议
7. 分镜重点

要求符合电影级视觉设计。
```

# 真正适用于所有项目的元模板

这是很多导演、AI视频创作者会长期保存的一版：

```text
请根据项目主题自动生成故事板镜头描述。

对于每个镜头必须输出：

【Scene】
时间
地点
环境

【Subject】
主体
外观
状态

【Action】
动作描述

【Camera】
景别
角度
运镜

【Composition】
构图方式

【Lighting】
灯光设计

【Mood】
情绪氛围

【Story Purpose】
镜头叙事目的

输出结果必须符合专业影视分镜标准，可直接用于 Storyboard、Midjourney、Flux、GPT Image、Runway、Veo、可灵、即梦 等视觉生成模型。
```

这样你只需要输入一句：

```text
项目主题：
暴雨中的侦探追踪嫌疑人
```

或者：

```text
项目主题：
高端腕表品牌广告
```

AI就能自动先生成完整的：

* 场景（Scene）
* 主体（Subject）
* 动作（Action）
* 镜头（Camera）
* 光线（Lighting）
* 情绪（Mood）
* 分镜目的（Story Purpose）

然后再进一步生成高质量故事板分镜图。这样比手工写场景、主体、动作更适合作为长期复用的通用模板。
