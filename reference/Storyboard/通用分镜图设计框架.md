
# 分镜图万能结构

以后无论是：

* 男装
* 女装
* 家具
* 汽车
* 美食
* 珠宝
* 手机
* 旅游
* 地产
* 品牌广告
* 剧情短片

全部套用：

```text
┌─────────────────────────┐
│                         │
│      镜头分镜区          │
│                         │
├─────────────────────────┤
│ 节奏 │ 运镜 │ 视觉说明   │
└─────────────────────────┘
```

---

# 真正通用的AI提示词

```text
请根据主题内容生成一张专业影视级分镜设计总览图（Storyboard Master Sheet）。

目标：

通过一张图完整展示视频的镜头规划、节奏规划、运镜规划和视觉规划。

版式结构：

第一部分：
镜头分镜区

采用网格布局展示完整镜头序列。

每个镜头包含：

- 镜头编号
- 景别
- 时长
- 时间码
- 缩略图预览
- 镜头说明
- 主体动作
- 画面重点

镜头按照视频叙事逻辑排序。

第二部分：
节奏设计区

展示：

- 视频时间轴
- 节奏变化
- 高潮位置
- 情绪曲线
- 音乐卡点

采用波形图或节奏图表现。

第三部分：
运镜设计区

展示：

- 摄影机机位
- 摄影机运动路线
- 摄影机方向
- 环绕路线
- 推拉摇移轨迹

采用俯视图方式绘制。

第四部分：
视觉设计区

展示：

- 灯光方案
- 色彩方案
- 构图方式
- 视觉关键词
- 氛围描述

采用设计说明形式呈现。

整体风格：

专业导演提案板
商业广告Storyboard
影视制作手册风格
品牌Campaign规划图
极简排版
信息图设计
高级灰白版式
专业制片级文档风格
超高清
```

# 如果想让AI自动适配任何题材

再加一层设计规则：

```text
镜头结构自动遵循：

建立
↓

介绍
↓

展示
↓

强化
↓

高潮
↓

收尾
```

```text

↓

↓

展示性能
↓

动态驾驶
↓

品牌收尾
```

# 最重要的一段（核心）

以后无论什么AI工具，都建议固定加入：

```text
Storyboard Master Sheet

Director Treatment Board

Creative Production Board

Shot List Layout

Camera Movement Diagram

Editing Rhythm Timeline

Visual Language System

Lighting & Mood Design

Professional Film Production Document

Advertising Campaign Planning Board

Information-rich storyboard presentation
```

这些词描述的是**设计形式**。

而：

```text
男装
汽车
家具
珠宝
```

只是替换的**内容主题**。

这样才是真正可复用的通用型分镜图模板。
