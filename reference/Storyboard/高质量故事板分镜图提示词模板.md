生成一张高质量故事板分镜图（Storyboard / Shot List），16:9横版，每个镜头单元独立、清晰。适用于电影、动画、短视频或广告项目。画面需呈现电影级视觉感和空间感，每个镜头包含以下要素：

1. 场景描述：
   - 具体环境（室内/室外/城市街道/自然景观/幻想场景等）
   - 背景物体和道具布局
   - 空间透视感（近景/远景/中景）

2. 构图与镜头：
   - 景别（ELS / LS / MLS / MS / MCU / CU / ECU）
   - 镜头角度（正面 / 俯拍 / 仰拍 / 侧身 / 环绕 / 特写）
   - 镜头运动（推 / 拉 / 平移 / 环绕 / 缓慢跟随 / 定镜）
   - 人物/主体位置（画面中心 / 左右 / 三分法构图）
   - 动作或表情（走动 / 跑动 / 转身 / 手势 / 表情特写）

3. 光线与氛围：
   - 主光、辅助光及方向（左前方 / 右后方 / 顶光 / 侧光）
   - 色温（暖色 / 冷色 / 中性）
   - 光质（柔光 / 硬光 / 逆光 / 透光）
   - 氛围（紧张 / 欢快 / 梦幻 / 沉重 / 神秘 / 浪漫）
   - 光影效果（阴影、轮廓光、体积光、反光）

4. 情绪与视觉风格：
   - 整体画面情绪（沉浸感 / 动感 / 空灵 / 压迫感）
   - 风格（写实 / 半写实 / 卡通 / 手绘 / CG / 黑白 / 高饱和）
   - 色彩搭配（单色 / 对比 / 暖冷色调混合）
   - 质感（柔软 / 硬朗 / 光滑 / 粗糙）

5. 文字信息：
   - 镜头编号与时间标注（例如：01 | MS 正面 | 0:00-0:03）
   - 镜头内容描述（简短一句话，突出动作或关键元素）
   - 可附额外备注（音效、配乐节奏或特效提示）

6. 画面表现要求：
   - 清晰可读的分镜板风格
   - 每个镜头独立且有占位空间
   - 空间透视准确
   - 真实或电影级光影效果
   - 高质量渲染感，便于导演或团队理解故事

示例提示词片段：
“室外黄昏城市街道，湿润地面反射街灯光，人物从左侧缓慢走向画面中心，中景，轻微侧身，手拿雨伞，表情沉思，主光来自右前方暖色街灯，背景有车流虚化，整体氛围梦幻、孤独，电影级写实风格。”

备注：
- 可根据项目需求自由增加或减少镜头单元数量
- 每个镜头保持一致视觉风格
- 场景、动作、光线、情绪等可替换不同主题
