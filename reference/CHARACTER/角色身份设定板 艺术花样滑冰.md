创作一幅艺术风格的 16:9 角色身份设定板（CHARACTER IDENTITY BOARD）。

[主题]：使用参考图像。艺术花样滑冰。无领设计。

纯白色/柔和的米白色背景。

无环境、无道具、无 Logo、无水印。

设计方向：

不要制作标准的角色参考图。

创作一张具有电影感的角色设定板，风格应类似于高端动画工作室的角色研究图与艺术画册排版的结合体。

布局应不对称、优雅且视觉上令人印象深刻。

使用大面积留白、变化的图像比例以及刻意的不平衡感。

避免网格、蓝图设计、目录式排版和重复性的转面展示。

重要布局规则：

任何角色图像不得重叠。

每个视角必须有清晰的间距和呼吸感。

保持所有全身像、肖像、剪影和细节研究在视觉上相互独立。

不裁剪面部、不隐藏肢体、不堆叠人物、不混合姿势。

主体构图：

将一个较大的英雄级全身像略微偏离中心放置，作为视觉锚点。

在其周围，以干净的间距排列较小的辅助研究图：

中性全身像、

背面视图、

侧面视图、

坐姿、

倚靠姿势、

蹲姿、

俯视身体角度、

仰视身体角度、

富有表现力的肖像研究。

每个视角都应像是一张独立的干净角色研究图，而非同一场景的连续帧。

身份锁定：

在所有视角中保持严格的身份一致性：

相同的面部、

相同的面部比例、

相同的发型、

相同的服装、

相同的身体比例、

相同的姿态语言、

相同的视觉个性。

实用参考细节：

使角色对未来的图像和视频生成具有可读性：

清晰的面部轮廓、

清晰的头发剪影、

清晰的服装剪影、

清晰的体型、

清晰的手部、

清晰的姿态、

清晰的表情范围。

艺术分区：

包含一个小型剪影研究区，带有 2-3 个简化的黑色角色剪影。

包含一个小型表情研究区，展示微妙的情绪变化。

包含一个小型细节研究区，展示面部、头发和服装的关键视觉特征。

文字设计：

添加一个时尚的 CHARACTER ID 信息块。

保持极简、粗体且具有艺术指导感。

仅使用：

NAME（姓名）

ROLE（角色定位）

CORE MOOD（核心氛围）

VISUAL SIGNATURE（视觉标识）

仅在必要时使用小型手写风格标签。

允许使用微妙的编辑箭头和注释标记，但需保持极简与优雅。

风格：

极简、

电影感、

高端、

艺术画册风格、

干净、

富有表现力、

适用于制作流程。

最终图像应像是一张艺术化的角色身份设定板，旨在帮助 AI 模型理解角色的面部、剪影、服装、姿态和情感范围。
